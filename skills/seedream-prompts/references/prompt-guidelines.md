# Seedream 4.0-4.5 提示词编写规范

> 基于火山引擎官方文档整理的提示词最佳实践。

## 一、五大通用规则详解

### 规则1：自然语言清晰描述画面

**原则**：用连贯的语句写明主体+行为+环境，无需零散堆砌词汇。

**推荐做法**：
```
一个身着华丽服饰的女孩，撑着遮阳伞漫步林荫道，莫奈油画风格
```

**避免做法**：
```
一个女孩，撑伞，林荫街道，油画般的细腻笔触
```

**解析**：
- ✅ 连贯语句，语义清晰
- ❌ 零散关键词堆砌，缺乏语境

---

### 规则2：明确图像应用场景与用途

**原则**：有具体使用场景时，直接标明图像类型和用途。

**推荐做法**：
```
设计游戏公司logo，主体是用游戏手柄打游戏的狗，logo标注公司名"PITBULL"
```

**避免做法**：
```
一张抽象图片，狗拿着游戏手柄，狗狗上写PITBULL
```

**常见应用场景关键词**：
| 场景 | 关键词 |
|------|--------|
| Logo设计 | logo、标志、品牌标识 |
| 海报设计 | 海报、宣传图、封面 |
| 图标设计 | 图标、icon、应用图标 |
| 角色设计 | 角色设定、人物设计、游戏角色 |
| 产品图 | 产品展示、电商图、商品图 |

---

### 规则3：精准提升风格渲染效果

**原则**：使用具体的风格关键词，或直接上传风格参考图。

**风格关键词对照表**：

| 风格类别 | 具体关键词 |
|----------|------------|
| 艺术流派 | 莫奈印象派、梵高风格、毕加索立体主义、浮世绘 |
| 摄影风格 | 电影感、纪实摄影、时尚摄影、商业摄影 |
| 数字风格 | 赛博朋克、蒸汽朋克、像素风、低多边形 |
| 传统风格 | 中国水墨、工笔画、浮世绘、油画 |
| 流行风格 | 日系清新、韩系唯美、欧美写实、二次元 |

**参考图使用**：
- 上传风格参考图能大幅提升画面贴合度
- 适用于复杂风格或独特美学要求

---

### 规则4：提高文字渲染准确度

**原则**：将需要生成的文字内容放在**双引号**内。

**推荐做法**：
```
生成一张海报，标题为"Seedream 4.5"
```

**避免做法**：
```
生成一张海报，标题为 Seedream 4.5
```

**文字渲染技巧**：
- 短文本效果更好（1-5个字最佳）
- 中英文混合时用引号分别包裹
- 复杂排版建议分步生成

---

### 规则5：清晰界定图片编辑要求

**原则**：用简洁指令说明修改对象和具体操作，避免模糊代词。

**推荐做法**：
```
让图中最高的熊猫穿上粉色京剧服饰并佩戴头饰，保持动作不变
```

**避免做法**：
```
让它穿上粉色衣服
```

**编辑指令结构**：
```
[修改对象] + [具体操作] + [保持要求]
```

**常见编辑操作**：
| 操作类型 | 示例 |
|----------|------|
| 增加 | 给图中女生增加同款银色耳线和项链 |
| 删除 | 去掉女生的帽子 |
| 替换 | 把最大的面包人换成牛角包形象，保持动作和表情不变 |
| 修改 | 让图中三个机器人变成透明水晶材质 |

## 二、视觉信号辅助编辑

当画面复杂时，使用涂鸦、线框等标记编辑位置，再搭配文字说明。

### 涂鸦标记

**示例**：
```
将房间内红色涂抹位置放入电视，蓝色涂抹位置放入沙发，保持原木风格和其他布局不变
```

### 线框标记

**示例**：
```
放大图中的标题至红框大小，改成和萨克斯图案一致的颜色与字体风格
```

## 三、参考图使用指南

### 参考图类型

| 参考类型 | 用途 | 示例 |
|----------|------|------|
| 人物形象 | 角色设计、人物手办 | 参考图中的人物形象，做一个动漫人物手办 |
| 艺术风格 | 风格迁移、统一视觉 | 参考图标的线性简约风格，设计应用图标 |
| 实体形象 | 材质转换、3D效果 | 将图中的形象转化为羊毛毡材质 |
| 产品款式 | 产品变体、系列设计 | 生成四件不同材质、不同颜色的上衣，款式一致 |
| 草图原型 | 草图转效果图 | 根据平面图生成实景图，遵循图中文字内容 |

### 参考图提示词公式

```
参考@图片的[特征类型]，[生成描述]
```

**示例**：
```
参考图中的人物形象，做一个动漫人物手办置于桌面，后方有印着角色图像的生日礼物包装盒
```

## 四、多图输入输出

### 多图输入

**核心技巧**：清晰指明不同参考图的编辑对象和具体操作。

**操作类型**：
| 操作 | 提示词示例 |
|------|------------|
| 替换 | 将图一的主体替换为图二的主体 |
| 组合 | 让图一人物穿上图二的服装 |
| 迁移 | 参考图二的艺术风格，对图一进行全画面风格转换 |

### 多图输出

**触发方式**：在提示词中加入"一系列"、"一套"、"组图"等词汇，或直接标注图片数量。

**适用场景**：
- 品牌视觉设计（VI系统）
- 影视分镜创作
- 成套日常设计（壁纸、表情包）
- IP产品设计

**示例**：
```
参考这个LOGO，做一套户外运动品牌"GREEN"的视觉设计，包含包装袋、帽子、卡片等周边，绿色主色调，简约现代风格
```
