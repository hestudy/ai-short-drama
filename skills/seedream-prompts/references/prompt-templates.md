## 一、文生图模板

### 万能公式

```
[场景描述] + [主体特征与动作] + [视觉风格] + [光影色调]
```

### 详细模板

```
[时间]的[地点]，[光线/天气]。
[主体描述]，[具体动作描述]，[情绪状态]。
[视觉风格]，[色调/光影]。
```

## 二、图生图模板

### 四大核心操作

#### 增加

```
给图中[对象]增加[添加内容]
```

**示例**：
```
给图中女生增加同款银色耳线和项链
```

#### 删除

```
去掉图中[对象]的[删除内容]
```

**示例**：
```
去掉女生的帽子
```

#### 替换

```
把图中的[原对象]换成[新对象]，保持[保持要求]
```

**示例**：
```
把最大的面包人换成牛角包形象，保持动作和表情不变
```

#### 修改

```
让图中的[对象]变成[变化描述]
```

**示例**：
```
让图中三个机器人变成透明水晶材质，从左到右为红黄绿三色，
绿色跑、黄色走、红色站立
```

---

### 视觉信号辅助模板

#### 涂鸦标记

```
将图中[颜色1]涂抹位置放入[内容1]，
[颜色2]涂抹位置放入[内容2]，
保持[保持要求]
```

**示例**：
```
将房间内红色涂抹位置放入电视，蓝色涂抹位置放入沙发，
保持原木风格和其他布局不变
```

#### 线框标记

```
放大图中的[目标]至[颜色]框大小，改成[变化描述]
```

**示例**：
```
放大图中的标题至红框大小，改成和萨克斯图案一致的颜色与字体风格
```

## 三、参考图生图模板

### 万能公式

```
参考@图片的[特征类型]，[生成描述]
```

---

### 参考人物形象

```
参考图中的人物形象，做一个动漫人物手办置于桌面，
后方有印着角色图像的生日礼物包装盒，盒下有书，盒前加圆形塑料底座，
手办站于底座上，室内场景，大片摄影感，手办在图左侧，尺寸与参考图一致。
```

---

### 参考艺术风格

```
参考图标的线性简约风格，设计[具体内容]，
保持配色统一。
```

**示例**：
```
参考图标的线性简约风格，设计音乐、天气、日历等9款应用图标，
保持配色统一。
```

---

### 参考实体形象

```
将图中的形象转化为[材质]，[附加描述]
```

**示例**：
```
将图中的形象转化为羊毛毡材质，搭配小支架维持原有姿势，放置在深色书桌上。
```

---

### 参考产品款式

```
生成[数量]件不同[变化维度]的[产品]，款式与图中[参考对象]一致，
仅展示[展示要求]。
```

**示例**：
```
生成四件不同材质、不同颜色的上衣，款式与图中女生的衣服一致，
仅展示衣服无模特。
```

---

### 草图转效果图

```
根据这张[草图类型]，生成[效果图描述]，
[风格要求]，布局和[参考内容]与例图一致，
[具体布局描述]，无手绘边缘和文字。
```

**示例**：
```
根据这张平面图，生成现代简约风精装客厅+开放式餐厅实景图，
地中海配色，布局和家具位置与例图一致，
餐桌区域阳光挑高，由近及远为沙发绿植、电视、餐桌椅子、落地窗，
无手绘边缘和文字。
```

## 五、多图输出模板

### 触发词汇

- "一系列"、"一套"、"组图"
- 直接标注数量："四张"、"七张"

---

### 品牌视觉设计

```
参考这个[参考对象]，做一套[品牌名]的视觉设计，
包含[内容列表]，[色调]主色调，[风格]风格。
```

**示例**：
```
参考这个LOGO，做一套户外运动品牌"GREEN"的视觉设计，
包含包装袋、帽子、卡片等周边，绿色主色调，简约现代风格。
```

---

### 影视分镜创作

```
生成[数量]张影视分镜图，依次为[分镜描述]，
画面连贯，[风格]风格。
```

**示例**：
```
生成四张影视分镜图，依次为宇航员在空间站维修飞船、遭遇陨石带袭击、
紧急躲避、受伤后惊险逃回飞船，画面连贯，电影镜头感。
```

---

### 成套日常设计

```
生成[时间范围][数量]张[内容描述]，
每张图[具体要求]，风格统一。
```

**示例**：
```
生成周一到周日七张自然景观手机壁纸，每张图清晰标注对应日期，风格统一。
```

---

### 角色换装

```
参考图中的人物形象，生成[数量]套不同[变化维度]的[角色]，
出现在[场景描述]，做不同的[动作描述]。
```

## 六、角色图片专用模板

### 角色外观提示词公式

```
[人物外貌] + [服装描述] + [表情姿态] + [风格] + [光影]
```

### 通用角色模板

```
一位[年龄]的[性别]，[发型发色]，[面部特征]，
穿着[服装描述]，[配饰描述]，
[表情]，[姿态/动作]，
[风格]风格，[光影]光影。
```

**示例**：
```
一位25岁的女性，黑色长发扎成低马尾，瓜子脸，大眼睛，
穿着白色职业衬衫和黑色西装外套，戴着细框眼镜，
表情严肃而自信，双手抱胸站立，
日系职场风格，柔和侧光。
```

### 情绪表达模板

| 情绪 | 表情描述 | 姿态描述 |
|------|----------|----------|
| 悲伤 | 眼眶含泪，嘴角微微下垂 | 低头，肩膀微微耸动 |
| 愤怒 | 眉头紧锁，眼神凌厉 | 双手握拳，身体前倾 |
| 喜悦 | 嘴角上扬，眼睛弯成月牙 | 步伐轻快，手势生动 |

| 沉思 | 目光游离，微微皱眉 | 手指轻触下巴，动作缓慢 |

## 八、常见问题解决

| 问题 | 解决方案 |
|------|----------|
| 人物面部不清晰 | 添加"面部清晰"、"高细节"、"4K画质" |
| 风格不统一 | 使用具体风格词，如"莫奈油画风格"、"赛博朋克" |
| 文字渲染错误 | 将文字用双引号包裹，如标题为"Seedream 4.5" |
| 编辑位置错误 | 使用涂鸦或线框标记位置 |
| 角色不一致 | 使用参考图 + "保持角色特征一致" |
| 多图风格不统一 | 添加"风格统一"、"色调一致" |
