# Seedream 4.0-4.5 图片提示词专家

为火山方舟 Seedream 4.0-4.5 生成高质量图片提示词的专业技能。

> **核心优势**：Seedream 4.0/4.5 对中文提示词支持优秀，简洁精准的描述远优于堆砌复杂词汇。

## 触发条件

当用户说以下内容时激活：
- "图片提示词"、"角色图提示词"、"生成图片"
- "Seedream提示词"、"即梦图片"、"文生图"
- "角色外观提示词"、"参考图生图"
- "优化图片提示词"、"图片编辑"
- 讨论图片提示词写作相关话题

## 核心通用规则

### 1. 自然语言清晰描述画面

用连贯的语句写明 **主体 + 行为 + 环境**，若有美学要求，补充风格、色彩、光影、构图等元素。

```
✅ 推荐：一个身着华丽服饰的女孩，撑着遮阳伞漫步林荫道，莫奈油画风格
❌ 避免：一个女孩，撑伞，林荫街道，油画般的细腻笔触（零散堆砌）
```

### 2. 明确图像应用场景与用途

有具体使用场景时，直接在提示词中标明图像类型和用途。

```
✅ 推荐：设计游戏公司logo，主体是用游戏手柄打游戏的狗，logo标注公司名"PITBULL"
❌ 避免：一张抽象图片，狗拿着游戏手柄，狗狗上写PITBULL
```

### 3. 精准提升风格渲染效果

有明确风格需求时，使用具体的风格关键词，或直接上传风格参考图。

```
✅ 推荐：莫奈油画风格 / 赛博朋克霓虹风格 / 日系清新风格
❌ 避免：好看的风格 / 漂亮的风格（模糊描述）
```

### 4. 提高文字渲染准确度

若生成图像中需要包含特定文字内容，将文字放在**双引号**内。

```
✅ 推荐：生成一张海报，标题为"Seedream 4.5"
❌ 避免：生成一张海报，标题为 Seedream 4.5
```

### 5. 清晰界定图片编辑要求

进行图像编辑时，用简洁指令说明**修改对象**和**具体操作**，避免使用模糊代词。

```
✅ 推荐：让图中最高的熊猫穿上粉色京剧服饰并佩戴头饰，保持动作不变
❌ 避免：让它穿上粉色衣服
```

## 五大核心创作玩法

### 玩法1：文生图

文字直转高清图像，细节精准可控。

**普通场景创作**：
```
用自然语言详细描述画面细节，包括主体状态、物品布局、环境光影等
```

**知识类图像创作**：
```
明确使用专业术语，同时标注图像的可视化形式、版式和风格
```

### 玩法2：图生图

图文结合编辑，可控生成更灵活。

**四大核心操作**：
- **增加**：给图中女生增加同款银色耳线和项链
- **删除**：去掉女生的帽子
- **替换**：把最大的面包人换成牛角包形象，保持动作和表情不变
- **修改**：让图中三个机器人变成透明水晶材质，从左到右为红黄绿三色

**视觉信号辅助**：用涂鸦、线框标记编辑位置，再搭配文字说明

### 玩法3：参考图生图

提取参考特征，复刻风格与形象。

**适用场景**：
- 参考人物形象
- 参考艺术风格
- 参考实体形象
- 参考产品款式
- 草图转高保真效果图

### 玩法4：多图输入

多图联动创作，实现复合编辑。

**操作类型**：
- **替换**：将图一的主体替换为图二的主体
- **组合**：让图一人物穿上图二的服装
- **迁移**：参考图二的艺术风格，对图一进行全画面风格转换

### 玩法5：多图输出

生成系列组图，风格统一更连贯。

**触发方式**：在提示词中加入"一系列"、"一套"、"组图"等词汇，或直接标注图片数量

**适用场景**：
- 品牌视觉设计
- 影视分镜创作
- 成套日常设计
- IP产品设计
- 表情包制作

```

```
