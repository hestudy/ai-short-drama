## 执行步骤

### 阶段0：项目初始化 (`/drama-new`)
1. 确认项目位置
2. 创建标准目录结构
3. 生成项目配置文件

**输出**: `00_项目配置.md` + 目录结构

### 阶段1：策划 (`/drama-plan`)
1. 收集需求（剧名、类型、集数、受众）
2. 生成故事大纲
3. 确定视觉风格

**输出**: `01_故事大纲.md`

### 阶段2A：角色设计 (`/drama-character`)
1. 确认角色清单
2. 创建角色设定卡
3. 定义一致性控制词

**输出**: `02_角色设定/角色设定卡.md`

### 阶段2B：场景设计 (`/drama-scene`) 🆕
1. 识别所有场景
2. 创建场景设定卡
3. 定义时段变体

**输出**: `04_场景设定/场景设定卡.md`

### 阶段3A：角色图提示词 (`/drama-image --character`)
1. 生成多角度提示词
2. 生成表情变体提示词
3. 在即梦AI生成参考图

**输出**: `03_角色图片/`

### 阶段3B：场景图提示词 (`/drama-image`)
1. 生成全景/中景/特写提示词
2. 生成时段变体提示词
3. 在即梦AI生成参考图

**输出**: `05_场景图片/`

### 阶段5：分镜脚本 (`/drama-storyboard`)
1. 选择生成模式（融合模式推荐）
2. 确定分镜策略
3. 逐集拆分镜头
4. 生成视频提示词

**输出**: `06_分镜脚本/` + `07_视频提示词/`

### 阶段6：视频提示词 (`/drama-prompt --batch`)
1. 批量生成视频提示词
2. 添加角色/场景参考图
3. 生成融合格式提示词

**输出**: `07_视频提示词/视频提示词清单.md`

### 阶段7-8：审核与补充
1. 补充缺失提示词
2. 审核一致性
3. 修正问题

**输出**: 审核报告

### 阶段9：导出 (`/drama-export`)
1. 汇总所有提示词
2. 生成制作清单

**输出**: `99_导出包/`

## 提示词规范

### Seedance 2.0 融合提示词结构
```
@角色参考图 @场景参考图

[主体描述]，[动作描述]，[情绪状态]。
[场景描述]，[时间天气]，[氛围]。
[视觉风格]，[色调光影]。
[镜头类型]，[运镜描述]。

配音：
- 角色：[说话角色]
- 声音：[声音特点]
- 台词：「[台词内容]」
- 情绪：[情绪]
- 语速：[语速]

音频：
- 环境音：[环境声音]
- 配乐：[背景音乐风格]

高清，4K，画面稳定，口型同步
```

### 角色图片提示词结构（/drama-image --character）
```
一位[年龄]的[性别]，[发型发色]，[面部特征]，
穿着[服装描述]，[配饰描述]，
[表情]，[姿态/动作]，
[风格]风格，[光影]光影。
```

### 场景图片提示词结构（/drama-image）
```
[场景描述]，[时间天气]。
[主要元素]，[空间层次]。
[视觉风格]，[色调光影]。
```

## 与即梦AI配合（v2.0）

### 完整工作流
1. **角色图**：`/drama-image --character` → 即梦AI → 保存到 `03_角色图片/`
2. **场景图**：`/drama-image` → 即梦AI → 保存到 `05_场景图片/`
3. **视频**：复制提示词（含 `@角色图 @场景图`）→ 即梦AI → 一键生成带配音视频
4. 剪辑软件整合

### 参考图命名规范
```
角色：@角色名_类型.jpg    例：@林小美_半身.jpg
场景：@场景名_视角.jpg    例：@天台_全景_黄昏.jpg
```

## 一致性控制（v2.0 核心特性）

### 双锚点控制
- **角色锚点**：`03_角色图片/` 中的成品图
- **场景锚点**：`05_场景图片/` 中的成品图

### 提示词引用
在视频提示词中使用 `@` 语法引用参考图：
```
@林小美_半身.jpg @天台_全景_黄昏.jpg

[提示词内容]
```

### 固定特征词
每个角色/场景都有固定的特征词，确保跨镜头一致性：
- 角色固定词：外貌、服装、标志性特征
- 场景固定词：空间布局、主要元素、色调
