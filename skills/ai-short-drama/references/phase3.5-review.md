# 阶段三·五：分镜与提示词审核

## 目标

审核分镜脚本与视频提示词的一致性，确保提示词准确还原分镜设计意图。

## 输入

- `03_分镜脚本/第X集.md`
- `04_视频提示词/第X集/`

## 输出

- `03_分镜脚本/第X集_审核通过.md`（审核通过后重命名）
- 审核报告（如有问题）

---

## 审核维度

### 0. 提示词完整性检查（优先）

> ⚠️ **第一步**：首先检查所有镜头是否都有提示词！

| 检查项 | 说明 | 通过标准 |
|--------|------|----------|
| 提示词存在 | 每个分镜都有对应的提示词 | 100%覆盖 |
| 提示词非空 | 提示词内容不为空或占位符 | 内容完整 |
| 提示词结构 | 包含必要的结构元素 | 场景+主体+动作+风格+运镜 |

**完整性检查模板：**
```markdown
# 提示词完整性检查

| 镜头 | 有分镜 | 有提示词 | 提示词长度 | 状态 |
|------|--------|----------|------------|------|
| S01_C01 | ✅ | ✅ | 120字 | ✅ |
| S01_C02 | ✅ | ✅ | 95字 | ✅ |
| S01_C03 | ✅ | ❌ | - | ⚠️ 缺失 |
| S01_C04 | ✅ | ✅ | 15字 | ⚠️ 过短 |

## 缺失/异常清单
- [ ] S01_C03 - 缺少提示词
- [ ] S01_C04 - 提示词过短，需补充
```

> 🚫 **阻断规则**：如果完整性检查未通过（有任何缺失或异常），必须先补充提示词，再进行后续审核！

### 1. 内容一致性

| 检查项 | 说明 | 通过标准 |
|--------|------|----------|
| 场景一致 | 提示词场景描述与分镜一致 | 无偏差 |
| 角色一致 | 提示词角色特征与角色卡一致 | 外观、服装无偏差 |
| 动作一致 | 提示词动作与分镜动作描述一致 | 关键动作完整呈现 |
| 时间一致 | 时间描述（清晨/夜晚等）一致 | 无冲突 |
| 情绪一致 | 提示词氛围与分镜情绪基调一致 | 情绪表达准确 |

### 2. 技术规范性

| 检查项 | 说明 | 通过标准 |
|--------|------|----------|
| 运镜匹配 | 提示词运镜与分镜运镜设计一致 | 景别、运镜方式匹配 |
| 时长合理 | 提示词内容量与镜头时长相符 | 不过长或过短 |
| 格式规范 | 符合即梦AI/Seedance 2.0 格式要求 | 结构完整 |

### 3. 连贯性

| 检查项 | 说明 | 通过标准 |
|--------|------|----------|
| 镜头衔接 | 相邻镜头能自然衔接 | 无突兀跳跃 |
| 角色状态 | 同一角色在不同镜头中状态连贯 | 外观、情绪连贯 |
| 场景连续 | 场景转换有逻辑 | 无空间跳跃 |

## 执行步骤

### Step 0: 提示词完整性检查（必须首先执行）

> ⚠️ 这是最重要的检查，必须在其他审核之前完成！

对每一集执行：

**0.1 统计检查**
```markdown
## 第X集完整性统计

- 分镜总数：N 个
- 有提示词数：M 个
- 缺失数：K 个
- 完整率：M/N × 100%
```

**0.2 逐镜头检查**
```
对每个镜头执行：
1. 检查分镜脚本中是否有该镜头
2. 检查提示词文件/章节中是否有对应提示词
3. 检查提示词内容是否完整（非空、非占位符）
4. 记录状态
```

**0.3 补充缺失提示词**

如果发现有镜头缺少提示词：

```markdown
## 需补充提示词的镜头

### S01_C03
**分镜信息：**
- 场景：[从分镜提取]
- 角色：[从分镜提取]
- 动作：[从分镜提取]
- 情绪：[从分镜提取]
- 运镜：[从分镜提取]

**补充的提示词：**
```
[根据分镜信息生成完整提示词]
```

**补充原因：** 原生成时遗漏
```

**0.4 完整性通过标准**

```
✅ 所有镜头都有对应的提示词
✅ 所有提示词内容非空
✅ 所有提示词结构完整（5要素齐全）
✅ 完整率 = 100%
```

> 🚫 只有通过完整性检查后，才能进入 Step 1 进行内容审核！

3. **人工修正**：复杂问题标记，由用户手动处理

### Step 4: 二次审核

修正后进行快速复核：


## 常见问题与修正

### 问题1：时间描述不一致
```
分镜：清晨的卧室
提示词：黄昏时分的卧室

修正：统一为"清晨"，调整光线描述为"柔和的晨光"
```

### 问题2：动作细节缺失
```
分镜：她缓慢地转身，眼神中带着失望
提示词：她转身

修正：添加"缓慢地"和"眼神中带着失望"
```

### 问题3：角色特征遗漏
```
分镜：穿着红色连衣裙的林小雨
提示词：一个女生

修正：补充"穿着红色连衣裙"和角色名引用
```

### 问题4：运镜不匹配
```
分镜：特写，缓慢推进
提示词：中景，固定镜头

修正：修改为"特写，镜头缓慢推进"
```

### 问题5：情绪氛围偏差
```
分镜：紧张不安的氛围
提示词：温馨的氛围

修正：修改氛围描述，调整光影为"高对比度，阴影深沉"
```
