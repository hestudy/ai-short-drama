# 阶段2B：场景设定工作流

## 概述

场景设定是AI短剧工作流的重要环节，与角色设定并行进行。良好的场景设定能确保：
- 视觉风格统一
- 场景转换流畅
- 情绪氛围准确
- 一致性可控

## 输入

- 故事大纲（`01_故事大纲.md`）
- 项目配置（`00_项目配置.md`）

## 输出

| 输出物 | 路径 | 说明 |
|--------|------|------|
| 场景设定卡 | `05_场景设定/[场景名]_设定卡.md` | 每个场景的详细设定 |
| 场景清单 | `05_场景设定/场景清单.md` | 所有场景的概览 |
| 图片提示词 | `06_场景图片/[场景名]_提示词.md` | 用于生成场景参考图 |

## 执行步骤

### Step 1: 场景识别

从故事大纲中识别所有场景：

```markdown
## 场景识别提示词

请分析以下故事大纲，识别出所有需要设计的场景：

[粘贴故事大纲]

输出格式：
| 序号 | 场景名 | 类型 | 出现时段 | 叙事功能 | 出现频率 |
|------|--------|------|----------|----------|----------|
| 1 | | 室内/室外 | | | 高/中/低 |
```

### Step 2: 场景优先级排序

根据出现频率和叙事重要性排序：

| 优先级 | 条件 | 处理顺序 |
|--------|------|----------|
| P0 | 主场景，频繁出现 | 最先设计 |
| P1 | 重要场景，关键情节 | 其次设计 |
| P2 | 次要场景，过渡用 | 可简化 |

### Step 3: 创建场景设定卡

为每个场景创建详细的设定卡，参考 `scene-card.md` 模板。

**关键要素**：

1. **空间描述**
   - 整体布局（开放/封闭、空间层次）
   - 主要元素（列出3-5个核心元素）
   - 背景层次（近景/中景/远景）
   - 标志性特征（视觉记忆点）

2. **氛围参数**
   - 时间与天气
   - 光线设定（光源、方向、强度、色温）
   - 色彩与情绪（主色调、饱和度、对比度）

3. **环境动态**
   - 自然动态（风、云、水流）
   - 人工动态（灯光、车流）
   - 声音元素（环境音）

4. **时段变体**
   - 白天样貌
   - 黄昏样貌
   - 夜晚样貌

5. **一致性控制**
   - 固定特征词
   - 禁止变化项

### Step 4: 生成图片提示词

使用 `/drama-image` 技能生成场景图片提示词：

```bash
# 全景空镜
/drama-image "[场景全景描述]" --style cinematic

# 中景视角
/drama-image "[场景中景描述]" --style cinematic

# 细节特写
/drama-image "[场景细节描述]" --style cinematic
```

**场景图片提示词公式**：
```
[场景描述] + [时间天气] + [主要元素] + [氛围情绪] + [视觉风格] + [光影色调]
```

### Step 5: 场景参考图生成

在即梦AI生成场景参考图：

1. 使用生成的提示词
2. 保存满意的成品图
3. 记录种子值
4. 归档到 `06_场景图片/` 目录

## 场景类型指南

### 室内场景

| 场景类型 | 空间特点 | 光线特点 | 常见氛围 |
|----------|----------|----------|----------|
| 居家 | 私密、温馨 | 柔和、温暖 | 舒适、安全 |
| 办公 | 开放、现代 | 明亮、冷调 | 专业、紧张 |
| 商业 | 装饰性强 | 聚焦、丰富 | 活力、消费欲 |
| 娱乐 | 昏暗、聚光 | 变化多端 | 放松、刺激 |

### 室外场景

| 场景类型 | 空间特点 | 光线特点 | 常见氛围 |
|----------|----------|----------|----------|
| 城市街道 | 线性、动态 | 多变 | 繁忙、孤独 |
| 自然风光 | 开阔、层次 | 自然 | 自由、宁静 |
| 公共空间 | 聚集、互动 | 混合 | 交流、观察 |
| 特殊地点 | 独特、标志 | 特色 | 情绪放大 |

### 虚构场景

| 场景类型 | 空间特点 | 光线特点 | 常见氛围 |
|----------|----------|----------|----------|
| 仙侠世界 | 飘逸、宏大 | 神秘、光效 | 超脱、震撼 |
| 科幻未来 | 几何、霓虹 | 人工、多彩 | 压迫、新奇 |
| 异世界 | 奇幻、想象 | 魔法光效 | 冒险、神秘 |
| 梦境 | 变形、流动 | 模糊、梦幻 | 潜意识、象征 |

## 一致性控制

### 固定特征词模板

每个场景应有一组固定特征词，在所有涉及该场景的提示词中出现：

```markdown
## 城市天台
固定特征词：
城市天台，高层建筑顶部，金属护栏，
远处城市天际线，高楼建筑群，开阔视野

## 咖啡馆
固定特征词：
小型咖啡馆室内，暖黄灯光，木质装饰，
落地窗，绿植点缀，温馨氛围
```

### 场景转换处理

场景转换时的注意事项：

1. **光线过渡**
   - 室内→室外：注意曝光变化
   - 白天→夜晚：注意色温变化
   - 自然光→人工光：注意质感变化

2. **情绪递进**
   - 场景与情绪匹配
   - 转换时有过渡镜头
   - 保留情绪连续性

3. **时间线一致**
   - 同一场景不同镜头时间连续
   - 避免时间跳跃过快

## 与角色设定的协调

场景设定与角色设定需要协调：

1. **风格统一**
   - 角色风格与场景风格一致
   - 避免角色与场景"格格不入"

2. **色调协调**
   - 角色服装与场景色调搭配
   - 考虑对比度和层次感

3. **比例合理**
   - 角色在场景中的比例协调
   - 空间关系合理
