# 即梦AI提示词模板库

## 一、视频生成提示词结构

### 基础公式
```
[场景描述] + [主体特征与动作] + [视觉风格] + [情绪或氛围] + [镜头语言]
```

### 详细模板
```
在[时间]的[地点]，[角色描述]正在[具体动作]，[动作细节描述]，[情绪状态]，[光影效果]，[视觉风格]，[镜头描述]
```

## 二、各类型场景模板

### 1. 室内日常

**卧室场景：**
```
[时间]的卧室，柔和的[光线颜色]光从窗帘透入，房间内[家具描述]，[角色描述][动作]，温馨/冷清的氛围，电影感构图
```

**办公室场景：**
```
现代化的办公室，[时间]的[光线]透过落地窗，办公桌[状态]，[角色描述][动作]，忙碌/压抑/轻松的氛围，中景构图
```

**咖啡厅场景：**
```
温馨的咖啡厅内部，暖色调灯光，背景有[细节]，[角色描述]坐在[位置][动作]，轻松的氛围，浅景深虚化背景
```

### 2. 室外场景

**街道场景：**
```
[城市类型]的街道，[时间][天气]，行人[状态]，[角色描述][动作]，[情绪氛围]，广角镜头，电影感
```

**公园场景：**
```
[季节]的公园，[时间]的阳光透过树叶，[环境细节]，[角色描述][动作]，[情绪氛围]，中景构图
```

**天台场景：**
```
城市高楼的屋顶，[时间]的天空[状态]，城市灯火[描述]，[角色描述][动作]，孤独/释然/期待的氛围，远景俯瞰
```

### 3. 特殊场景

**回忆/闪回：**
```
[场景描述]，柔和的过曝效果，画面边缘略微模糊，[角色描述][动作]，怀旧温暖的氛围，轻微晃动的手持感
```

**梦境/幻想：**
```
[超现实场景描述]，梦幻的光影效果，色彩饱和度高，[角色描述][动作]，神秘/奇幻的氛围，缓慢的运镜
```

**紧张/悬疑：**
```
[场景描述]，阴影重，明暗对比强烈，[角色描述][动作]，紧张不安的氛围，快速剪辑感，特写镜头
```

## 三、角色动作模板

### 情绪动作

**悲伤：**
```
眼中含泪，嘴角微微颤抖，缓慢地低下头，肩膀微微耸动
```

**愤怒：**
```
眉头紧锁，眼神凌厉，紧紧握拳，呼吸急促
```

**喜悦：**
```
嘴角上扬，眼睛弯成月牙，步伐轻快，手势生动
```

**惊讶：**
```
眼睛睁大，嘴巴微张，身体微微后仰，动作停滞
```

**沉思：**
```
目光游离，微微皱眉，手指轻触下巴，动作缓慢
```

### 日常动作

**走路：**
```
[情绪状态]地走在[地点]，脚步[轻快/沉重/匆忙]，[手部动作]，[头部动作]
```

**坐下：**
```
[情绪状态]地[坐姿]坐下，身体[前倾/后仰/端正]，[手部动作]
```

**看手机：**
```
低头看着手机屏幕，手指[滑动/打字/点击]，表情[描述]
```

**对话：**
```
面向[对方]，表情[描述]，手势[描述]，嘴唇[开合状态]
```

## 四、运镜模板

### 基础运镜

**推进：**
```
镜头从[起始位置]缓慢/快速推进至[终点位置]
```

**拉远：**
```
镜头从[起始位置]缓慢/快速拉远至[终点位置]
```

**环绕：**
```
镜头围绕[主体]360度/180度环绕，[速度]运动
```

**跟随：**
```
镜头跟随[主体]从[起点]移动到[终点]，保持[距离]
```

**升降：**
```
镜头从[高度]缓慢/快速升至/降至[高度]
```

### 情绪运镜

**紧张感：**
```
手持镜头轻微晃动，快速推近角色面部特写
```

**抒情：**
```
缓慢的推拉镜头，配合角色情绪变化
```

**震撼：**
```
大范围拉远，展示[宏大场景]
```

**亲密：**
```
温柔的推进，聚焦角色面部微表情
```

## 六、完整示例

### 示例1：都市情感

**镜头：** 女主下班回家

```
黄昏时分的城市街道，夕阳将天空染成橙红色，街道两旁亮起霓虹灯。一位25岁左右的职场女性，穿着职业套装，背着公文包，疲惫地走在人行道上，偶尔抬头看看天，眼神中透露着孤独和迷茫。温暖的橙色调，电影感构图，镜头从侧面中景跟随她缓慢移动。
```

### 示例2：悬疑紧张

**镜头：** 男主发现线索

```
深夜的办公室，只有台灯发出冷白色的光，在墙上投下深深的阴影。一位30岁左右的男性侦探，眉头紧锁，盯着桌上的照片，手指颤抖地拿起照片，眼睛睁大，瞳孔微缩。紧张悬疑的氛围，高对比度光影，镜头缓慢推进至他面部特写。
```

### 示例3：温馨日常

**镜头：** 情侣做饭

```
明亮的开放式厨房，周末上午的阳光从窗户洒入，厨房台面上摆满食材。一对年轻情侣站在一起准备做饭，男生在切菜，女生在旁边指导，两人相视而笑，温馨甜蜜的氛围。暖色调，柔和光线，中景固定镜头。
```

## 七、提示词优化技巧

### 增强效果的关键词

| 效果 | 关键词 |
|------|--------|
| 提升画质 | 高清画质、电影级、4K、细腻 |
| 增强氛围 | 氛围感、电影感、质感、层次丰富 |
| 强化情绪 | 情绪饱满、眼神有戏、表情细腻 |
| 优化光影 | 光影层次、自然光线、戏剧性用光 |

### 负面提示词

```
--no blurry（避免模糊）
--no low resolution（避免低分辨率）
--no static image（避免静态图片感）
--no cartoonish（避免卡通感，如需写实）
--no distorted（避免变形）
--no watermark（避免水印）
```

### 权重调整

```
[关键词:1.5]  # 增强该元素权重
[关键词:0.8]  # 降低该元素权重
```
