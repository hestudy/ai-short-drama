# 阶段二：角色设计工作流

## 目标

为所有主要角色创建完整的设定卡，包括可直接用于即梦AI的图片生成提示词。

## 输入

- `01_故事大纲.md`

## 输出

- `02_角色设定/[角色名].md` × N
- `02_角色设定/角色外观提示词.md`

---

## 执行步骤

### Step 1: 角色清单确认

基于故事大纲，列出需要详细设计的角色：

```
根据故事大纲，需要设计以下角色：

【主角】
- 角色A

【主要配角】
- 角色B
- 角色C

【次要配角】
- 角色D
- 角色E

【反派】
- 角色F

请确认是否需要增减角色？
```

### Step 2: 逐个设计角色

对每个角色，按以下流程：

#### 2.1 基础信息确认

```
【角色名】设计：

1. 年龄：
2. 职业：
3. 性格关键词（3-5个）：
4. 在剧中的功能：
```

#### 2.2 外貌描述生成

生成符合即梦AI的图片提示词：

```markdown
## 外貌描述

### 基础描述
[性别]，[年龄段]，[体型]，[整体气质]

### 面部特征
[脸型]，[肤色]，[眼睛]，[发型发色]，[特殊标记]

### 服装造型
[常规服装]，[配饰]

### 即梦AI提示词

**正面标准照：**
```
[完整描述]，正面视角，中性表情，[光线]，高清画质，[风格]
```

**侧面照：**
```
[完整描述]，侧面视角，[光线]，高清画质，[风格]
```

**表情参考-喜悦：**
```
[完整描述]，开心微笑，眼睛明亮，[光线]，高清画质，[风格]
```

**表情参考-悲伤：**
```
[完整描述]，悲伤表情，眼眶微红，[光线]，高清画质，[风格]
```

**表情参考-愤怒：**
```
[完整描述]，愤怒表情，眉头紧锁，[光线]，高清画质，[风格]
```
```

#### 2.3 角色深度设计

```markdown
## 性格特征

- 核心特质：...
- 行为习惯：...
- 口头禅：...

## 背景故事

- 成长经历：...
- 关键事件：...
- 内心秘密：...

## 角色弧线

- 起点状态：...
- 转变契机：...
- 终点状态：...

## 标志性元素

- 标志性动作：...
- 标志性台词：...
- 标志性道具：...
```

### Step 3: 汇总角色外观提示词

创建 `02_角色设定/角色外观提示词.md`：

```markdown
# 角色外观提示词汇总

> 复制到即梦AI生成角色参考图

---

## [角色A]

### 正面标准照
```
[提示词]
```

### 侧面照
```
[提示词]
```

### 表情参考
```
喜悦：[提示词]
悲伤：[提示词]
愤怒：[提示词]
```

---

## [角色B]

（同上格式）

---

## 使用说明

1. 复制上述提示词到即梦AI图片生成
2. 生成满意的形象后保存作为"角色种子图"
3. 后续生成视频时上传此图保持角色一致性
```

---

## 完成标准

- [ ] 所有主要角色都有完整的设定卡
- [ ] 每个角色都有可用的即梦AI图片提示词
- [ ] 角色提示词汇总文件已创建
- [ ] 用户已确认角色设定
