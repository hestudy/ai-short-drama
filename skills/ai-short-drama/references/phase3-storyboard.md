# 阶段三：分镜脚本工作流

## 目标

将故事转化为具体的分镜脚本，并为每个镜头生成即梦AI视频提示词。

> 💡 **Seedance 2.0 融合模式**：支持在提示词中直接融合配音信息，一键生成带配音的视频！

## 输入

- `01_故事大纲.md`
- `02_角色设定/`

## 输出

- `03_分镜脚本/第X集.md` × N
- `04_视频提示词/第X集/` × N

---

## ⚠️ 并发控制配置

> 🔒 **重要**：本阶段是调用频率最高的环节，必须严格遵守并发限制！

| 参数 | 值 | 说明 |
|------|-----|------|
| **批次大小** | 5 个镜头/批 | 每批处理的镜头数量 |
| **并发数** | 3 | 同时进行的最大请求数 |
| **间隔时间** | 1000ms | 批次之间的等待时间 |
| **建议模式** | 批量顺序处理 | 平衡质量和效率 |

**预估调用次数对比：**

| 方式 | 50个镜头调用次数 | 说明 |
|------|------------------|------|
| 逐个处理 | 100 次 | 每镜头2次调用（分镜+提示词） |
| 批量处理 | 10 次 | 每批5个，共10批 |
| **节省** | **90%** | 大幅降低限流风险 |

详细配置参见：[concurrency-control.md](./concurrency-control.md)

## 执行步骤

### Step 1: 选择生成模式

与用户确认：

```
生成模式选择：

1. 【Seedance 2.0 融合模式】（推荐）
   - 提示词包含画面+配音+音频
   - 一键生成带配音视频
   - 无需后续对口型
   - 适合：有台词的剧情短剧

2. 【传统分步模式】
   - 仅生成画面提示词
   - 后续需要单独配音
   - 适合：无声画面/后期单独配音

请选择生成模式（1/2）：
```

### Step 2: 确定分镜策略

与用户确认：

```
分镜策略选择：

1. 【镜头密度】
   - 紧凑型：每分钟约8-10个镜头
   - 标准型：每分钟约5-7个镜头
   - 舒缓型：每分钟约3-5个镜头

2. 【叙事风格】
   - 对话驱动：多中近景，强调人物互动
   - 动作驱动：多动感镜头，强调运动感
   - 氛围驱动：多远景特写切换，强调情绪渲染

3. 【镜头时长偏好】
   - 快节奏：平均2-3秒/镜头
   - 中等：平均4-5秒/镜头
   - 慢节奏：平均6秒以上/镜头

4. 【提示词模式】（🆕 新增）
   - 标准模式：每个镜头独立生成，适合精细控制
   - 时间分段模式：15秒内按时间段精确控制，适合连贯动作场景
   - 单镜头模式：一镜到底的长镜头，适合追逐/运动场景

请选择或说明您的偏好：
```

**提示词模式说明：**

| 模式 | 适用场景 | 优势 | 示例 |
|------|----------|------|------|
| **标准模式** | 对话、情感、日常场景 | 精确控制每个镜头 | 每个镜头4-6秒独立生成 |
| **时间分段模式** | 穿越变身、动作爆发、情绪递进 | 一个镜头内完成多个阶段 | 0-3秒开场 → 3-8秒变化 → 8-15秒高潮 |
| **单镜头模式** | 飞车追逐、跑酷、长镜头追踪 | 展示运动连贯性 | "Single continuous shot..." |

## 🆕 时间分段模式详解

> 适用于需要在一个较长镜头（如15秒）内精确控制多个动作阶段的场景

### 适用场景

- ✅ 穿越变身场景（现代→古代）
- ✅ 动作爆发场景（蓄力→出击→爆发）
- ✅ 情绪递进场景（平静→发现→震惊）
- ✅ 运动竞技场景（对峙→交锋→结果）
- ✅ 魔法/特效场景（起势→施法→爆发）

### 时间分段模板

**3段式（推荐，易于控制）：**
```
15秒，[整体风格]

0-5秒：[场景建立/起始动作]
镜头：[镜头1描述]

5-10秒：[动作发展/冲突升级]
镜头：[镜头2描述]

10-15秒：[高潮/结尾/揭示]
镜头：[镜头3描述]

音效：[音频描述]
限制：[负面提示]
```

**5段式（精细控制）：**
```
15秒，[整体风格]

0-3秒：[开场镜头]
3-6秒：[动作A]
6-9秒：[动作B]
9-12秒：[动作C]
12-15秒：[收尾/反转]

音效：[音频描述]
限制：[负面提示]
```

### 完整示例

**示例1：穿越剧开场**
```
15秒，中国古装短剧风格，电影画质，暖金色与冷青色交替，2.35:1宽屏压缩至9:16竖屏

0-3秒：现代都市夜景，一位身穿白色连衣裙的年轻女孩站在天桥上低头看手机，突然脚下一滑
镜头：中景，平视，轻微晃动

3-8秒：女孩失重坠落，身体在空中旋转，周围景物扭曲变形，光线拖尾
镜头：环绕跟随，快速旋转，光影流动

8-15秒：画面骤然稳定，女孩落在古代宫殿的花园中，周围樱花飞舞，她惊讶地站起身，低头看自己的衣服已变成古装
镜头：仰拍全景推进至中景，樱花特写

音效：都市车流声 → 坠落风声 → 古琴轻响
限制：无模糊，人物面部清晰
```

**示例2：篮球盖帽**
```
15秒，电影写实风格，黄金时刻暖光，美国街头篮球氛围

0-3秒：洛杉矶户外街头篮球场，阳光斜照，一位高大肌肉发达的美国黑人男球员穿着无袖训练背心，正在做交叉运球热身，肌肉线条分明，表情自信轻蔑。对面站着一位娇小的美国金发高马尾女孩，穿着宽松篮球短袖，表情淡定从容。男球员微笑着拍球示意她来防守。
镜头：中景，平视对峙

4-8秒：男球员开始进攻，连续炫技式交叉运球和背后运球，球鞋摩擦地面扬起灰尘。他突然加速向右突破，准备上篮。手持跟拍镜头紧贴他的脚步移动。女孩动作迅速灵活，侧身滑步卡位，表情专注，双眼紧盯着篮球。
镜头：手持跟拍，低角度

9-12秒：慢动作加速，男球员跳起准备右手上篮。就在球举过头顶的瞬间，女孩同时起跳，身体舒展腾空。她的右手在最高点精准有力地拍在篮球上，球旋转着飞出画面。男球员空中的表情瞬间从自信变为震惊，眼睛睁大嘴巴张开。
镜头：仰拍，捕捉两人腾空瞬间

13-15秒：恢复正常速度，男球员落地踉跄后退两步，双手撑膝弯腰喘气，满脸不可置信。女孩轻松落地，甩一下马尾辫，面无表情地捡起篮球，单手运两下，微微歪头瞥他一眼带着一丝轻蔑的笑，转身离开，背影干脆利落。
镜头：全景固定，女孩背影远去

音效：街头环境声贯穿始终，清晰的运球声和球鞋吱吱声。盖帽瞬间沉闷有力的拍击声，伴随周围观众「Ohhh」的惊叹声。结尾以低沉有节奏的嘻哈节拍收尾。
限制：无文字、无字幕、无水印
```

### 时间分段模式注意事项

1. **时间分配要合理**：确保每个阶段有足够时间展示
2. **镜头衔接要自然**：阶段之间的转换要流畅
3. **情绪递进要清晰**：从开场到高潮要有明显节奏
4. **总时长建议15秒**：Seedance 2.0 的最佳时长

## 限流恢复策略

如果遇到 API 限流：

1. **暂停处理**：停止当前批次
2. **指数退避**：
   - 第一次等待 2 秒
   - 第二次等待 4 秒
   - 最大等待 60 秒
3. **调整配置**：
   - 将批次大小从 5 降至 3
   - 将间隔时间从 1000ms 增至 2000ms
4. **重试**：重新发送请求
