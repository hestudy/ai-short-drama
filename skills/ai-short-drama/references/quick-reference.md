## 快速开始

### 1分钟创建项目

```bash
/drama-new 我的短剧 --episodes 5
/drama-plan
/drama-character --all
/drama-scene --all
```

### 生成参考图

```bash
# 角色图
/drama-image --character "25岁女性，干练自信" --style cinematic

# 场景图
/drama-image "城市天台，黄昏" --style cinematic
```

### 生成视频提示词

```bash
/drama-storyboard 1
/drama-prompt --batch 06_分镜脚本/第1集_分镜.md
```

## 一致性控制

### 双锚点
- **角色锚点**：`03_角色图片/` 中的成品图
- **场景锚点**：`05_场景图片/` 中的成品图

### @引用语法
```
@林小美_半身.jpg @天台_全景_黄昏.jpg

[提示词内容]
```

### 固定特征词
每个角色/场景都有一组固定特征词，在所有提示词中出现：

**角色**：年龄、性别、发型、标志性特征
**场景**：空间布局、主要元素、色调

## 提示词公式

### 角色图
```
一位[年龄]的[性别]，[发型发色]，[面部特征]，
穿着[服装]，[配饰]，
[表情]，[姿态]，
[风格]风格，[光影]光影。
```

### 场景图
```
[场景描述]，[时间天气]。
[主要元素]，[空间层次]。
[视觉风格]，[色调光影]。
```

### 视频提示词（融合模式）
```
@角色图 @场景图

[主体]，[动作]，[情绪]。
[场景]，[时间]，[氛围]。
[风格]，[色调]。
[镜头]，[运镜]。

配音：
- 角色：xxx
- 台词：「xxx」
- 情绪：xxx

音频：
- 环境音：xxx
- 配乐：xxx

高清，4K，画面稳定
```

## 常见问题

### Q: 角色在不同镜头中不一致怎么办？
A: 确保每次生成时：
1. 使用相同的固定特征词
2. 使用角色参考图（@语法）
3. 记录满意的种子值

### Q: 场景色调不统一怎么办？
A: 在场景设定卡中明确定义：
1. 固定特征词
2. 色调参数（暖调/冷调）
3. 使用场景参考图

### Q: 如何快速迭代？
A: 使用批量命令：
```bash
/drama-character --all
/drama-scene --all
/drama-prompt --batch
```

## 文件命名规范

### 角色参考图
```
@角色名_类型.jpg
例：@林小美_半身.jpg, @林小美_全身.jpg, @林小美_特写.jpg
```

### 场景参考图
```
@场景名_视角_时段.jpg
例：@天台_全景_黄昏.jpg, @咖啡馆_中景.jpg
```

### 分镜文件
```
第X集_分镜.md
例：第1集_分镜.md
```

### 视频提示词
```
第X集_视频提示词.md
例：第1集_视频提示词.md
```
