# Seedance 2.0 视频提示词专家

为 Seedance 2.0 生成高质量视频提示词的专业技能。

> **核心发现**：经过实测验证，Seedance 2.0 对中文提示词的支持**优于英文**。中文专业术语（如"俯拍"、"跟拍"、"环绕拍"）的识别度和精准度更高。

## 触发条件

当用户说以下内容时激活：
- "生成提示词"、"写提示词"、"提示词生成"
- "Seedance提示词"、"即梦提示词"、"视频提示词"
- "优化提示词"、"改进提示词"
- "时间分段提示词"、"长镜头提示词"
- 讨论视频提示词写作相关话题

## 核心原则

### 1. 优先使用中文

```
✅ 推荐：俯拍、跟拍、环绕拍、推镜、拉镜
❌ 避免：top-down shot, tracking shot, orbit shot（效果较差）
```

### 2. 必加质量词

**每条提示词必须包含以下质量词**：
```
高清，4K，画面稳定，无抖动，细节丰富，面部清晰，无变形
```

### 3. 动作描述具体连续

```
❌ 模糊："一个人在走路"
✅ 具体："一位25岁女性，穿着职业套装，疲惫地走在黄昏的街道上，脚步缓慢，偶尔抬头看天"
```

### 4. 场景+动作+镜头组合

```
[场景描述] + [主体特征与动作] + [视觉风格] + [镜头语言] + [质量词]
```

## 提示词模式

### 1. 标准模式（推荐新手）

适用于大多数场景，简洁高效：

```
[场景描述]，[时间/天气/光线]。
[主体描述]，[具体动作描述]，[情绪/状态]。
[视觉风格]，[色调/光影]。
[镜头类型]，[运镜描述]。
[质量词]
```

### 2. 融合模式（含配音）

适用于需要配音的场景，画面+音频一体化：

```
[场景描述]，[时间/天气/光线]。
[主体描述]，[具体动作描述]，[情绪/状态]。
[视觉风格]，[色调/光影]。
[镜头类型]，[运镜描述]。

配音：
- 角色：[说话角色]
- 声音：[声音特点]
- 台词：「[台词内容]」
- 情绪：[情绪描述]
- 语速：[语速]

音频：
- 环境音：[背景声音]
- 配乐：[音乐风格]

[质量词]
```

### 3. 时间分段模式

适用于连贯动作场景，15秒内精确控制：

```
[总时长]，[整体风格]

[时间段1]：[画面描述] + [动作描述] + [镜头描述]
[时间段2]：[画面描述] + [动作描述] + [镜头描述]
[时间段3]：[画面描述] + [动作描述] + [镜头描述]

音效：[音频描述]
限制：[负面提示]
[质量词]
```

### 4. 单镜头模式

适用于展示运动能力的长镜头：

```
单镜头连续拍摄，[主体描述]
[动作轨迹描述]
镜头[运镜描述]
最终揭示[画面揭示]
[质量词]
```

## 中文专业术语

### 镜头术语

| 中文 | 效果 | 适用场景 |
|------|------|----------|
| 俯拍 | 从上往下拍摄 | 展示全貌、渺小感 |
| 仰拍 | 从下往上拍摄 | 崇高感、压迫感 |
| 跟拍 | 镜头跟随主体移动 | 行走、追逐、对话 |
| 环绕拍 | 镜头围绕主体旋转 | 展示人物、英雄时刻 |
| 推镜 | 镜头向前推进 | 强调、聚焦、紧张 |
| 拉镜 | 镜头向后拉远 | 揭示、展示、结尾 |
| 固定镜头 | 镜头不动 | 沉稳、观察、对话 |
| 手持镜头 | 轻微晃动 | 紧张、真实、临场感 |

### 景别术语

| 中文 | 描述 | 适用场景 |
|------|------|----------|
| 大远景 | 展示环境全貌 | 场景建立、史诗感 |
| 远景 | 展示场景与人物关系 | 人物与环境互动 |
| 全景 | 人物全身 | 动作展示、走路 |
| 中景 | 人物膝盖以上 | 对话、日常 |
| 近景 | 人物胸部以上 | 情感表达 |
| 特写 | 人物面部 | 情绪细节 |
| 大特写 | 眼睛、嘴唇等局部 | 极强情绪冲击 |

### 风格术语

| 中文 | 描述 |
|------|------|
| 电影感 | 戏剧性光影、高对比 |
| 复古胶片风 | 暖色调、颗粒感、柔和边缘 |
| 赛博朋克 | 霓虹灯光、蓝紫色调、未来感 |
| 日系清新 | 柔和光线、高调色彩、浅景深 |
| 古风唯美 | 水墨意境、暖色调、飘逸感 |
| 黑色电影 | 高对比、阴影深沉、冷色调 |

## 场景类型速查

| 类型 | 关键词 | 典型动作 |
|------|--------|----------|
| 日常 | 温馨、生活化 | 起床、做饭、工作 |
| 情感 | 情绪饱满、特写 | 表白、争吵、拥抱 |
| 动作 | 动感、快节奏 | 打斗、追逐、运动 |
| 科幻 | 霓虹、未来感 | 机甲、太空、特效 |
| 奇幻 | 梦幻、光效 | 魔法、龙、超自然 |
| 古装 | 水墨、古典 | 武侠、宫廷、穿越 |
| 悬疑 | 阴影、高对比 | 发现、对峙、揭秘 |

## 参考素材语法

```
@图片1 作为首帧
@图片2 作为角色外观参考
@视频1 作为镜头运镜参考
@视频2 作为动作参考
@音频1 用于生成配乐
```

**限制**：最多 9 张图片 + 3 段视频 + 3 段音频，总计 ≤12 个文件

## 质量检查清单

### 必须检查
- [ ] 包含明确的主体
- [ ] 动作描述具体连续
- [ ] 镜头类型和运动清晰
- [ ] 风格和氛围一致
- [ ] 质量词完整（高清、4K、画面稳定、无抖动、细节丰富、面部清晰）

### 建议检查
- [ ] 光线/色调描述完整
- [ ] 情绪传达清晰
- [ ] 时长合理（4-15秒）
- [ ] 负面提示（如需要）
- [ ] 参考素材正确引用
