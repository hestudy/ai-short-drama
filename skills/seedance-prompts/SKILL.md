为 Seedance 2.0 生成高质量视频提示词的专业技能。

## 触发条件

当用户说以下内容时激活：
- "生成提示词"、"写提示词"、"提示词生成"

## 提示词模式

### 1. 标准模式

适用于大多数场景，每个镜头独立生成：

```
Subject: [主体描述]
Action: [动作描述]
Camera: [运镜描述]
Style: [风格描述]

Voice: (如需配音)
- 角色：[说话角色]
- 声音：[声音特点]
- 台词：「[台词内容]」
- 情绪：[情绪]
- 语速：[语速]

Audio: (如需音频)
- 环境音：[环境声音]
- 配乐：[背景音乐风格]
- 音效：[特殊音效]

Constraints: [约束条件]
```

### 2. 时间分段模式

适用于连贯动作场景（15秒内精确控制）：

```
15秒，[整体风格]

[时间1]：[画面+动作+镜头]
[时间2]：[画面+动作+镜头]
[时间3]：[画面+动作+镜头]

音效：[音频描述]
限制：[负面提示]
```

### 3. 单镜头模式

适用于展示运动能力的长镜头：

```
Single continuous shot, [主体描述],
[动作轨迹描述],
camera [镜头运动],
final reveal [最终揭示]
```

## 提示词公式

### 基础公式
```
[场景描述] + [主体特征与动作] + [视觉风格] + [情绪或氛围] + [镜头语言]
```

### 融合公式（含配音）
```
[画面描述] + [配音指令] + [音频氛围]
```

### 时间分段公式
```
[总时长]，[整体风格]
[时间段]：[画面] + [动作] + [镜头]
```

## 场景类型

| 类型 | 子类型 | 关键词 |
|------|--------|--------|
| 日常 | 卧室、办公室、咖啡厅、街道 | 温馨、写实、生活化 |
| 情感 | 表白、争吵、和解、告白 | 情绪饱满、特写、氛围 |
| 动作 | 打斗、追逐、运动、竞技 | 动感、快速剪辑、慢动作 |
| 科幻 | 未来都市、太空、机甲 | 霓虹、赛博朋克、粒子效果 |
| 奇幻 | 魔法、龙、超自然 | 梦幻、光效、史诗 |
| 古装 | 武侠、宫廷、穿越 | 水墨、古典、飘逸 |
| 悬疑 | 发现、追逐、对峙 | 阴影、高对比、紧张 |

## 运镜模式

### 基础运镜
| 类型 | 描述 | 适用场景 |
|------|------|----------|
| 推进 | 镜头向前靠近 | 强调、聚焦 |
| 拉远 | 镜头向后退 | 揭示、展示 |
| 环绕 | 镜头围绕主体 | 展示、人物 |
| 跟随 | 镜头跟随移动 | 行走、追逐 |
| 升降 | 镜头上下移动 | 高度变化 |

### 情绪运镜
| 情绪 | 运镜方式 |
|------|----------|
| 紧张 | handheld shake, fast push to close-up |
| 震撼 | wide pull-back reveal of massive scene |
| 亲密 | slow gentle push to face close-up |
| 史诗 | 360 orbit, rising crane shot, wide angle |
| 悬疑 | static shot, sudden rack focus/zoom |

## 参考素材语法

```
@图片1 作为首帧
@图片2 作为角色外观参考
@视频1 作为镜头运镜参考
@视频2 作为动作参考
@音频1 用于生成配乐
```

**限制**：最多 9 张图片 + 3 段视频 + 3 段音频，总计 ≤12 个文件
