# 即梦AI提示词模板库

## 一、视频生成提示词结构

### 基础公式
```
[场景描述] + [主体特征与动作] + [视觉风格] + [情绪或氛围] + [镜头语言]
```

### 详细模板
```
在[时间]的[地点]，[角色描述]正在[具体动作]，[动作细节描述]，[情绪状态]，[光影效果]，[视觉风格]，[镜头描述]
```

## 二、各类型场景模板

### 1. 室内日常

**卧室场景：**
```
[时间]的卧室，柔和的[光线颜色]光从窗帘透入，房间内[家具描述]，[角色描述][动作]，温馨/冷清的氛围，电影感构图
```

**办公室场景：**
```
现代化的办公室，[时间]的[光线]透过落地窗，办公桌[状态]，[角色描述][动作]，忙碌/压抑/轻松的氛围，中景构图
```

**咖啡厅场景：**
```
温馨的咖啡厅内部，暖色调灯光，背景有[细节]，[角色描述]坐在[位置][动作]，轻松的氛围，浅景深虚化背景
```

### 2. 室外场景

**街道场景：**
```
[城市类型]的街道，[时间][天气]，行人[状态]，[角色描述][动作]，[情绪氛围]，广角镜头，电影感
```

**公园场景：**
```
[季节]的公园，[时间]的阳光透过树叶，[环境细节]，[角色描述][动作]，[情绪氛围]，中景构图
```

**天台场景：**
```
城市高楼的屋顶，[时间]的天空[状态]，城市灯火[描述]，[角色描述][动作]，孤独/释然/期待的氛围，远景俯瞰
```

### 3. 特殊场景

**回忆/闪回：**
```
[场景描述]，柔和的过曝效果，画面边缘略微模糊，[角色描述][动作]，怀旧温暖的氛围，轻微晃动的手持感
```

**梦境/幻想：**
```
[超现实场景描述]，梦幻的光影效果，色彩饱和度高，[角色描述][动作]，神秘/奇幻的氛围，缓慢的运镜
```

**紧张/悬疑：**
```
[场景描述]，阴影重，明暗对比强烈，[角色描述][动作]，紧张不安的氛围，快速剪辑感，特写镜头
```

## 三、角色动作模板

### 情绪动作

**悲伤：**
```
眼中含泪，嘴角微微颤抖，缓慢地低下头，肩膀微微耸动
```

**愤怒：**
```
眉头紧锁，眼神凌厉，紧紧握拳，呼吸急促
```

**喜悦：**
```
嘴角上扬，眼睛弯成月牙，步伐轻快，手势生动
```

**惊讶：**
```
眼睛睁大，嘴巴微张，身体微微后仰，动作停滞
```

**沉思：**
```
目光游离，微微皱眉，手指轻触下巴，动作缓慢
```

### 日常动作

**走路：**
```
[情绪状态]地走在[地点]，脚步[轻快/沉重/匆忙]，[手部动作]，[头部动作]
```

**坐下：**
```
[情绪状态]地[坐姿]坐下，身体[前倾/后仰/端正]，[手部动作]
```

**看手机：**
```
低头看着手机屏幕，手指[滑动/打字/点击]，表情[描述]
```

**对话：**
```
面向[对方]，表情[描述]，手势[描述]，嘴唇[开合状态]
```

## 四、运镜模板

### 基础运镜

**推进：**
```
镜头从[起始位置]缓慢/快速推进至[终点位置]
```

**拉远：**
```
镜头从[起始位置]缓慢/快速拉远至[终点位置]
```

**环绕：**
```
镜头围绕[主体]360度/180度环绕，[速度]运动
```

**跟随：**
```
镜头跟随[主体]从[起点]移动到[终点]，保持[距离]
```

**升降：**
```
镜头从[高度]缓慢/快速升至/降至[高度]
```

### 情绪运镜

**紧张感：**
```
手持镜头轻微晃动，快速推近角色面部特写
```

**抒情：**
```
缓慢的推拉镜头，配合角色情绪变化
```

**震撼：**
```
大范围拉远，展示[宏大场景]
```

**亲密：**
```
温柔的推进，聚焦角色面部微表情
```

## 九、时间分段式提示词（高级）

> 💡 **核心技巧**：将15秒视频按时间段精确控制，每个时间段定义独立的画面、动作和镜头

### 基础公式

```
[总时长]，[整体风格/氛围]

[时间1]：[画面描述] + [角色动作] + [镜头]
[时间2]：[画面描述] + [角色动作] + [镜头]
[时间3]：[画面描述] + [角色动作] + [镜头]
...

音效：[音频描述]
限制：[负面提示]
```

### 时间分段模板

**3段式（5秒×3 = 15秒）：**
```
15秒，[整体风格]

0-5秒：[场景建立/起始动作]，[镜头1]
5-10秒：[动作发展/冲突升级]，[镜头2]
10-15秒：[高潮/结尾/揭示]，[镜头3]
```

**5段式（3秒×5 = 15秒）：**
```
15秒，[整体风格]

0-3秒：[开场镜头]，[镜头1]
3-6秒：[动作A]，[镜头2]
6-9秒：[动作B]，[镜头3]
9-12秒：[动作C]，[镜头4]
12-15秒：[收尾/反转]，[镜头5]
```

## 十、电影级运镜模板

### 单镜头长镜头（One Continuous Shot）

> 🎬 单镜头是展示 Seedance 2.0 运动理解能力的最佳方式

**基础公式：**
```
Single continuous shot, [主体] + [动作轨迹], camera [镜头运动], final reveal [最终揭示]
```

**示例1：飞车追逐**
```
Single continuous shot, speeder chase through narrow streets of a Mediterranean village,
camera weaves through hanging laundry and market stalls, rider barely dodges obstacles,
final reveal opens to massive cliff edge overlooking infinite ocean at golden hour
```

**示例2：悬崖城市飞车（详细版）**
```
From a monumental cliffside city carved into stone, the camera dives toward a tiny streak of light
ripping along a narrow ledge-road. Lock-on: a speeder hugging the wall at insane speed.
The camera slingshots ahead, whips back, then drops tight to the rear thrusters:
heat haze, grit snapping off the ledge, warning lights flashing.
A collapsing balcony rains debris; the rider snaps a last-inch swerve under a falling arch,
then threads through hanging laundry lines and open windows in one fluid line.
The camera darts through the same openings, staying glued to the motion.
One final bend and sudden calm: the camera blasts outward into a reveal of the city
opening onto a boundless waterfall-fed valley, mist turning into rainbow.
```

**示例3：摩托追逐**
```
Single continuous shot, motorcycle chase through narrow streets of a Mediterranean village,
camera weaves through hanging laundry and market stalls, rider barely dodges obstacles,
final reveal opens to massive cliff edge overlooking infinite ocean at golden hour
```

### 动作场景运镜

**战斗场景：**
```
Two martial artists fighting on a rooftop at sunset,
spinning kicks and rapid punches, wind blowing clothes,
smooth cinematic camera orbit, slow motion mid-air kick, epic vibe
```

**紧张感运镜：**
```
handheld camera with subtle shake, fast push to close-up on character face,
shallow depth of field, rack focus between subject and threat
```

**史诗感运镜：**
```
wide pull-back reveal showing massive [场景],
camera rises from ground level to aerial view,
volumetric lighting, lens flare
```

### 情绪运镜对照表

| 情绪 | 运镜方式 | 示例 |
|------|----------|------|
| 紧张 | 手持晃动 + 快速推近 | handheld shake, fast push to close-up |
| 震撼 | 大范围拉远揭示 | wide pull-back reveal of massive scene |
| 亲密 | 缓慢推进聚焦 | slow gentle push to face close-up |
| 史诗 | 环绕 + 升降 + 广角 | 360 orbit, rising crane shot, wide angle |
| 悬疑 | 固定 + 突然变焦 | static shot, sudden rack focus/zoom |
| 浪漫 | 柔和跟随 + 浅景深 | soft follow shot, shallow depth of field |

## 十三、物理真实感增强

> 💡 解决AI视频中常见的"漂浮感"、"失重感"、"缺乏惯性"问题

### 真实感关键词

```
hyperrealistic physics, accurate weight distribution,
natural inertia and momentum, realistic gravity,
proper body mechanics, authentic movement weight
```

### 运动场景优化

**追逐/战斗场景：**
```
添加：realistic weight, natural momentum, proper inertia
避免：floating, weightless, unnatural movement
```

**人物动作：**
```
添加：grounded movement, natural body mechanics, realistic force transfer
避免：puppet-like motion, mechanical movement
```

## 十五、常见问题解决

| 问题 | 解决方案 |
|------|----------|
| 人物面部模糊 | 添加"面部清晰"、"4K"、"高清画质" |
| 动作不自然 | 添加"自然动作"、"流畅"、"真实物理" |
| 镜头太静态 | 明确镜头运动："缓慢推进"、"环绕"、"跟随" |
| 风格不统一 | 使用具体风格词："赛博朋克"、"黑色电影" |
| 人物漂浮 | 添加"真实重力"、"自然惯性"、"地面接触" |
| 情绪不到位 | 明确情绪词 + 表情/肢体描述 |
| 配音不同步 | 使用融合格式，明确"口型同步" |
