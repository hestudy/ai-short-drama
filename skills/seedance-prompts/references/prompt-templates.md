# Seedance 2.0 提示词模板库

> **核心原则**：Seedance 2.0 对中文提示词的支持**优于英文**，本模板库优先使用中文专业术语。

## 一、提示词基础公式

### 万能公式

```
[场景描述] + [主体特征与动作] + [视觉风格] + [镜头语言] + [质量词]
```

### 详细模板

```
[时间]的[地点]，[光线/天气]。
[主体描述]，[具体动作描述]，[情绪状态]。
[视觉风格]，[色调/光影]。
[镜头类型]，[运镜描述]。
[质量词]
```

### 质量词（必加）

**每条提示词必须包含**：
```
高清，4K，画面稳定，无抖动，细节丰富，面部清晰，无变形
```

**按需添加**：
- 动作流畅、无卡顿
- 色彩自然、无偏色
- 口型同步（配音场景）
- 表情同步（配音场景）

### 2. 融合模式（4-15秒）

**适用场景**：对话、独白、情感戏、配音场景

> ⚠️ **重要**：Seedance 2.0 支持原生音频生成，可在视频生成时同步配音！

```
[场景描述]，[时间/天气/光线]。
[主体描述]，[具体动作描述]，[情绪/状态]。
[视觉风格]，[色调/光影]。
[镜头类型]，[运镜描述]。

配音：
- 角色：[说话角色]
- 声音：[声音特点]
- 台词：「[台词内容]」
- 情绪：[情绪描述]
- 语速：[语速]

音频：
- 环境音：[背景声音]
- 配乐：[音乐风格]
- 音效：[特殊音效]

[质量词]
```

**配音字段说明**：

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 角色 | ✅ | 说话的角色标识 | 女主角、旁白、男主 |
| 声音 | ✅ | 声音特点描述 | 温柔女声、低沉男声、活泼少女 |
| 台词 | ✅ | 具体台词内容（用「」包裹） | 「我明白了」 |
| 情绪 | ⬜ | 情绪基调 | 悲伤、愤怒、平静、激动 |
| 语速 | ⬜ | 说话速度 | 偏快、正常、偏慢、逐字说出 |

### 3. 时间分段模式（10-15秒）

**适用场景**：穿越、变身、动作爆发、情节递进

```
[总时长]，[整体风格]

[时间段1]：[画面描述] + [动作描述] + [镜头描述]
[时间段2]：[画面描述] + [动作描述] + [镜头描述]
[时间段3]：[画面描述] + [动作描述] + [镜头描述]

音效：[音频描述]
限制：[负面提示]
[质量词]
```

**时间段划分建议**：
- 3段式：0-5秒、5-10秒、10-15秒
- 5段式：0-3秒、3-6秒、6-9秒、9-12秒、12-15秒

**示例1：穿越剧开场**
```
15秒，中国古装短剧风格，电影画质，暖金色与冷青色交替

3-8秒：女孩失重坠落，身体在空中旋转，周围景物扭曲变形，光线拖尾
镜头：环绕跟拍，快速旋转，光影流动

8-15秒：画面骤然稳定，女孩落在古代宫殿的花园中，周围樱花飞舞，她惊讶地站起身，低头看自己的衣服已变成古装
```

**示例2：篮球盖帽**
```

0-3秒：洛杉矶户外街头篮球场，阳光斜照，一位高大肌肉发达的美国黑人男球员穿着无袖训练背心，正在做交叉运球热身。对面站着一位娇小的美国金发高马尾女孩，穿着宽松篮球短袖，表情淡定从容

音效：街头环境声，运球声，盖帽拍击声，观众惊叹声
```


**动作复刻**：
```
参考@视频1的人物动作，
生成@图片1的角色和@图片2的角色的[场景描述]，
[动作细节]，[氛围描述]
```

**风格迁移**：
```

[场景描述]，极致的[美学风格]
```

## 八、物理真实感增强

> 解决AI视频中常见的"漂浮感"、"失重感"、"缺乏惯性"问题

### 真实感关键词

```
真实物理效果，重量分布准确，自然惯性和动量，真实重力
```

### 运动场景优化

**追逐/战斗场景**：
```
添加：真实重量感，自然动量，适当惯性
避免：漂浮，失重，不自然运动
```

**人物动作**：
```
添加：接地运动，自然身体力学，真实力量传递
避免：木偶般运动，机械运动
```

## 十、常见问题解决

| 问题 | 解决方案 |
|------|----------|
| 人物面部模糊 | 添加"面部清晰"、"4K"、"高清画质" |
| 动作不自然 | 添加"自然动作"、"流畅"、"真实物理效果" |
| 镜头太静态 | 明确镜头运动："缓慢推进"、"环绕拍"、"跟拍" |
| 风格不统一 | 使用具体风格词："赛博朋克"、"黑色电影"、"古风唯美" |
| 人物漂浮 | 添加"真实重力"、"自然惯性"、"接地运动" |
| 情绪不到位 | 明确情绪词 + 表情/肢体描述 |
| 配音不同步 | 使用融合模式，明确"口型同步" |
