# 场景类型详解

> **核心原则**：场景描述要具体，包含时间、光线、环境细节，让AI更精准理解画面需求。

## 一、室内日常场景

### 卧室场景

**氛围关键词**：
- 温馨、私密、舒适
- 清晨阳光、夜晚月光
- 柔和光线、温暖色调

**提示词模板**：
```
[时间]的卧室，柔和的[光线颜色]光从窗帘透入，
房间内[家具描述]，床铺[状态]，
[角色描述][动作]，温馨的氛围，
[视觉风格]，中景
[质量词]
```

**典型动作**：
- 起床、赖床、伸懒腰
- 看手机、发呆
- 更衣、化妆
- 思考、回忆

**示例**：
```

房间内整洁温馨，床铺有些凌乱，
一位25岁的年轻女性穿着睡衣坐在床边，双手抱着膝盖，眼神空洞地望着窗外，
温馨略带忧郁的氛围，电影感构图，暖黄色调，
中景固定，缓慢推向面部特写。
高清，4K，画面稳定无抖动，面部清晰
```
