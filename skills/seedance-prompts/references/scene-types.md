# 场景类型详解

## 一、日常场景

### 卧室场景

**氛围关键词**：
- 温馨、私密、舒适
- 清晨阳光、夜晚月光
- 柔和光线、温暖色调

**提示词模板**：
```
[时间]的卧室，柔和的[光线颜色]光从窗帘透入，
房间内[家具描述]，床铺[状态]，
[角色描述][动作]，温馨的氛围，
电影感构图，中景
```

**典型动作**：
- 起床、赖床
- 看手机
- 更衣、化妆
- 思考、发呆

---

### 办公室场景

**氛围关键词**：
- 专业、忙碌、紧张
- 荧光灯、电脑屏幕光
- 中性色调、冷色调

**提示词模板**：
```
现代化的办公室，[时间]的[光线]透过落地窗，
办公桌[状态]，文件[摆放]，
[角色描述][动作]，[忙碌/压抑/轻松]的氛围，
中景构图
```

**典型动作**：
- 工作、开会
- 电话、打字
- 思考、压力
- 休息、交流

---

### 咖啡厅场景

**氛围关键词**：
- 轻松、温暖、社交
- 暖色灯光、木质装饰
- 咖啡香气、轻音乐

**提示词模板**：
```
温馨的咖啡厅内部，暖色调灯光，
背景有[细节：书架/绿植/其他顾客]，
[角色描述]坐在[位置][动作]，
轻松的氛围，浅景深虚化背景
```

**典型动作**：
- 喝咖啡、聊天
- 工作、看书
- 等待、观察
- 思考、写作

---

### 街道场景

**氛围关键词**：
- 繁忙、流动、城市感
- 自然光、霓虹灯
- 车流、人流

**提示词模板**：
```
[城市类型]的街道，[时间][天气]，
行人[状态]，车辆[状态]，
[角色描述][动作]，[情绪氛围]，
广角镜头，电影感
```

**典型动作**：
- 行走、奔跑
- 等待、张望
- 交谈、争吵
- 思考、回忆

---

## 二、情感场景

### 表白场景

**氛围关键词**：
- 紧张、期待、浪漫
- 柔和光线、暖色调
- 私密空间、背景虚化

**提示词模板**：
```
[场景描述]，[光线描述]，
[角色A]站在[角色B]面前，表情[紧张/期待]，
[动作描述：握住手/深情注视]，
浪漫氛围，面部特写推进
```

**关键要素**：
- 眼神交流
- 肢体接触
- 情绪递进
- 环境配合

---

### 争吵场景

**氛围关键词**：
- 紧张、激烈、情绪爆发
- 高对比度、阴影
- 快速剪辑、手持感

**提示词模板**：
```
[场景描述]，[光线：暗调/高对比]，
[角色A]与[角色B]面对面站立，
[动作描述：指手画脚/摔东西]，
紧张激烈的氛围，快速剪辑，手持镜头
```

**关键要素**：
- 情绪对比
- 空间距离变化
- 动作张力
- 声音设计

---

### 和解场景

**氛围关键词**：
- 温暖、释然、感动
- 柔和光线、温暖色调
- 缓慢运镜、特写

**提示词模板**：
```
[场景描述]，[光线：温暖柔和]，
[角色A]与[角色B]相视，
[动作描述：拥抱/握手/微笑]，
温暖释然的氛围，缓慢推进至特写
```

---

## 三、动作场景

### 打斗场景

**氛围关键词**：
- 紧张、激烈、快节奏
- 动态光线、高对比
- 快速剪辑、手持

**提示词模板**：
```
Two [角色类型] fighting in [场景],
[动作描述：出拳/踢腿/闪避],
[特效描述：尘土飞扬/光效],
手持跟拍，快速剪辑，慢动作高潮
```

**运镜建议**：
- 手持晃动增加临场感
- 慢动作突出关键动作
- 快速剪辑增加节奏

---

### 追逐场景

**氛围关键词**：
- 紧张、快速、危险
- 动态光线、运动模糊
- 跟随镜头、穿越镜头

**单镜头模式**：
```
Single continuous shot, [主体] chase through [场景],
camera weaves through [障碍物],
final reveal [最终揭示]
```

---

### 运动竞技

**氛围关键词**：
- 激烈、热血、竞争
- 户外光线、运动场灯光
- 跟随镜头、慢动作

**时间分段模式**：
```
15秒，电影写实风格，[运动氛围]

0-5秒：对峙/准备，中景平视
5-10秒：交锋/动作，手持跟拍
10-15秒：结果/反应，全景慢动作
```

---

## 四、特殊场景

### 科幻场景

**未来都市**：
```
Subject: 未来都市天际线，悬浮建筑群，全息广告牌
Action: 飞行器穿梭于摩天大楼之间
Camera: 大范围航拍，穿越建筑间隙
Style: 赛博朋克风格，霓虹灯光，蓝紫色调，高对比度
```

**太空场景**：
```
Subject: 巨大的太空站，玻璃穹顶
Action: 镜头穿过气闸进入站内
Camera: 第一人称视角，推进镜头
Style: 科幻电影质感，星光闪烁，冷色调
```

**机甲战斗**：
```
Subject: 巨型机甲战士
Action: 变形、挥拳、能量爆发
Camera: 低角度仰拍，快速切换
Style: 真人电影级特效，金属质感，粒子效果
```

---

### 奇幻场景

**魔法对决**：
```
Subject: 两位魔法师
Action: 施放对立元素魔法，蓝与金能量碰撞
Camera: 环绕拍摄，定格爆炸瞬间
Style: 好莱坞奇幻大片，粒子特效，魔法光效
```

**龙之战场**：
```
Subject: 武士与巨型飞龙
Action: 在浮空岩石岛屿上追逐战斗
Camera: 快速穿梭于岛屿之间，手持感
Style: 超现实奇幻，暴风雨背景，闪电效果
```

---

### 古装场景

**武侠对决**：
```
Subject: 两位古装剑客
Action: 剑光交错，内力外放，招式对决
Camera: 慢动作定格，快速剪辑
Style: 武侠电影质感，水墨意境，飘逸感
```

**宫廷场景**：
```
Subject: 古代宫殿内
Action: 人物缓慢行走，衣摆飘动
Camera: 跟随镜头，优雅平稳
Style: 古典美学，暖色调，柔和光线
```

---

### 悬疑场景

**发现真相**：
```
Subject: 角色 + 关键线索物品
Action: 震惊表情，颤抖的手，瞳孔收缩
Camera: 缓慢推近面部特写，焦点转移
Style: 黑色电影风格，高对比度，阴影深沉
```

**追逐戏**：
```
Subject: 追逐者与被追者
Action: 穿越狭窄通道，翻越障碍
Camera: 手持跟拍，快速剪辑感
Style: 紧张氛围，暗色调，不稳定构图
```

---

## 五、场景转换

### 闪回/回忆

**提示词**：
```
[场景描述]，柔和的过曝效果，
画面边缘略微模糊，
[角色描述][动作]，
怀旧温暖的氛围，轻微晃动的手持感
```

**关键词**：
- overexposed glow
- soft vignette edges
- warm vintage color grading
- subtle handheld shake
- nostalgic atmosphere

---

### 梦境/幻想

**提示词**：
```
[超现实场景描述]，
梦幻的光影效果，色彩饱和度高，
[角色描述][动作]，
神秘/奇幻的氛围，缓慢的运镜
```

**关键词**：
- surreal landscape
- dreamlike lighting
- saturated colors
- floating elements
- ethereal atmosphere

---

### 时间流逝

**提示词**：
```
[固定场景]，光线快速变化，
[动作：日出到日落/四季更替]，
延时摄影效果，时间流逝感
```

**关键词**：
- time-lapse
- shifting light
- day to night transition
- seasonal change
