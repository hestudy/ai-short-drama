# AI短剧文字工作流

使用 Claude Code 完成 AI 短剧的全部文字创作工作，输出可直接用于即梦AI等平台的文本内容。

## 触发方式

- 用户说："AI短剧"、"写短剧"、"短剧创作"、"生成短剧提示词"时自动激活
- 手动调用：`/ai-short-drama`

## 核心功能

### 1. 项目初始化
```
/ai-short-drama new [剧名] --episodes [集数]
```
创建项目目录结构，初始化项目文件。

### 2. 策划阶段
```
/ai-short-drama plan
```
- 生成故事大纲
- 设计核心冲突
- 规划剧情弧线
- 设定目标受众

### 3. 角色设计
```
/ai-short-drama character [角色名]
```
- 创建角色设定卡
- 生成角色外观描述（可用于即梦AI图片生成）
- 设计角色性格特征
- 规划角色发展弧线

### 4. 分镜脚本
```
/ai-short-drama storyboard [集数]
```
- 拆解每集为具体镜头
- 为每个镜头生成完整描述
- 规划镜头语言和运镜

### 5. 提示词生成
```
/ai-short-drama prompt [镜头编号]
```
- 生成即梦AI视频生成提示词
- 结构化：场景 + 主体 + 动作 + 风格 + 情绪
- 支持批量生成整集提示词

### 6. 配音台本
```
/ai-short-drama script [集数]
```
- 生成配音台本文本
- 标注情绪和语速要求
- 支持多角色对话

### 7. 导出
```
/ai-short-drama export
```
- 导出所有提示词清单
- 生成即梦AI可直接使用的文本包

## 提示词生成规范

### 视频生成提示词结构
```
[场景描述], [主体特征与动作], [视觉风格], [情绪或氛围], [镜头语言]
```

### 角色图片提示词结构
```
[人物外貌], [服装描述], [表情姿态], [风格], [光影]
```

### 场景图片提示词结构
```
[环境描述], [时间天气], [氛围], [风格], [构图]
```

## 工作流程

1. **初始化项目** → 创建目录结构
2. **故事大纲** → 确定核心概念和剧情走向
3. **角色设计** → 创建角色卡 + 外观描述
4. **分镜脚本** → 拆解每集为具体镜头
5. **生成提示词** → 为每个镜头生成即梦AI提示词
6. **配音台本** → 生成配音文本
7. **导出** → 打包所有可直接使用的内容

## 与即梦AI的配合

1. 在 Claude Code 中完成所有文字创作
2. 导出提示词和台本
3. 复制到即梦AI平台：
   - 角色外观提示词 → 生成角色参考图
   - 场景提示词 → 生成场景参考图
   - 视频提示词 → 生成视频片段
   - 配音台本 → AI配音
4. 在剪辑软件中整合

## 注意事项

- 提示词使用中文，符合即梦AI的理解习惯
- 每个提示词控制在100字以内，简洁精准
- 保持角色描述的一致性
- 标注每个提示词的用途（角色/场景/视频）
