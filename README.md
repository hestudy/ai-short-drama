# AI 短剧文字工作流

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-2.2.0-blue.svg)](https://github.com/hestudy/ai-short-drama)

> Claude Code Plugin - 在 Claude Code 中完成 AI 短剧的全部文字创作工作

## 简介

这是一个 Claude Code 插件，用于系统化地完成 AI 短剧的文字创作工作。生成的提示词和台本可直接用于即梦AI等平台进行视频制作。

**🔥 v2.2.0 重大更新**：
- **场景设定工作流** 🆕：新增 `/drama-scene` 命令，完整的场景设计流程
- **双锚点一致性控制** 🆕：角色+场景双锚点，确保视频生成一致性
- **新目录结构**：标准化输出目录，角色图和场景图分离管理
- **技能深度集成**：角色图/场景图/视频提示词独立生成，流程更清晰

## 安装

### 方式一：插件市场安装（推荐）

在 Claude Code 中运行以下命令：

```bash
# 添加插件市场
/plugin marketplace add hestudy/ai-short-drama

# 安装插件
/plugin install ai-short-drama@ai-short-drama
```

### 方式二：克隆到用户级目录

```bash
git clone https://github.com/hestudy/ai-short-drama.git ~/.claude/plugins/ai-short-drama
```

### 方式三：项目级安装

```bash
mkdir -p your-project/.claude/plugins
git clone https://github.com/hestudy/ai-short-drama.git your-project/.claude/plugins/ai-short-drama
```

## 命令列表

| 命令 | 说明 | 阶段 |
|------|------|------|
| `/drama-new` | 创建新项目 | 阶段0 |
| `/drama-plan` | 策划故事大纲 | 阶段1 |
| `/drama-character` | 设计角色设定卡 | 阶段2A |
| `/drama-scene` 🆕 | 设计场景设定卡 | 阶段2B |
| `/drama-image` | 图片提示词专家（角色/场景） | 阶段3A/3B |
| `/drama-storyboard` | 生成分镜脚本 | 阶段5 |
| `/drama-prompt` | 视频提示词专家 | 阶段6 |
| `/drama-fill-prompts` | 检查并补充缺失的提示词 | 阶段7 |
| `/drama-review` | 审核分镜与提示词一致性 | 阶段8 |
| `/drama-script` | 生成配音台本（传统模式） | - |
| `/drama-export` | 导出全部提示词包 | 阶段9 |

### 快速开始

```bash
# 1. 创建项目
/drama-new "我的短剧" --episodes 5

# 2. 生成故事大纲
/drama-plan

# 3. 创建设定（并行）
/drama-character --all
/drama-scene --all

# 4. 生成参考图（在即梦AI中）
/drama-image --character "角色描述"
/drama-image "场景描述"

# 5. 生成分镜和视频提示词
/drama-storyboard 1
/drama-prompt --batch

# 6. 导出
/drama-export
```

## 一致性控制（核心特性）⭐

### 双锚点控制

| 锚点 | 目录 | 用途 |
|------|------|------|
| **角色锚点** | `03_角色图片/` | 确保角色跨镜头一致 |
| **场景锚点** | `05_场景图片/` | 确保场景跨镜头一致 |

### @引用语法

在视频提示词中引用参考图：

```
@林小美_半身.jpg @天台_全景_黄昏.jpg

黄昏时分的城市天台，夕阳将天空染成橙红色。
一位25岁的女孩，穿着白色连衣裙，独自站在护栏边...
```

### 固定特征词

每个角色/场景都有一组固定特征词：

**角色**：
```
25岁亚洲女性，黑色长发及腰，瓜子脸杏眼，
左眼角泪痣，纤细身材，...
```

**场景**：
```
城市天台，高层建筑顶部，金属护栏，
远处城市天际线，高楼建筑群，开阔视野
```

## 提示词规范

### 角色图片提示词（/drama-image --character）

```
一位[年龄]的[性别]，[发型发色]，[面部特征]，
穿着[服装描述]，[配饰描述]，
[表情]，[姿态/动作]，
[风格]风格，[光影]光影。
```

### 场景图片提示词（/drama-image）

```
[场景描述]，[时间天气]。
[主要元素]，[空间层次]。
[视觉风格]，[色调光影]。
```

### 视频提示词（融合模式）

```
@角色参考图 @场景参考图

[主体描述]，[动作描述]，[情绪状态]。
[场景描述]，[时间天气]，[氛围]。
[视觉风格]，[色调光影]。
[镜头类型]，[运镜描述]。

配音：
- 角色：[说话角色]
- 声音：[声音特点]
- 台词：「[台词内容]」
- 情绪：[情绪]
- 语速：[语速]

音频：
- 环境音：[环境声音]
- 配乐：[背景音乐风格]

高清，4K，画面稳定，口型同步
```

### 示例

**角色图**：
```
一位25岁的女性，黑色长发扎成低马尾，瓜子脸，大眼睛，
穿着白色职业衬衫和黑色西装外套，戴着细框眼镜，
表情严肃而自信，双手抱胸站立。
电影感风格，柔和侧光，高细节，面部清晰。
```

**场景图**：
```
城市高层建筑天台，黄昏时分，
夕阳将天空染成橙红色，远处城市天际线轮廓清晰。
金属护栏，开阔视野，微风吹动。
电影感构图，暖色调，浅景深，4K。
```

**视频提示词（融合模式）**：
```
@林小美_半身.jpg @天台_全景_黄昏.jpg

黄昏时分的城市天台，夕阳将天空染成橙红色。
一位25岁的女孩，穿着白色连衣裙，独自站在护栏边，
望向远方的城市天际线，发丝随风轻轻飘动。
电影感构图，温暖的橙色调，柔和光线。
镜头缓慢推近，从全景过渡到中景。

配音：
- 角色：林小美
- 声音：温柔女声
- 台词：「我等你很久了。」
- 情绪：期待/悲伤
- 语速：缓慢

音频：
- 环境音：风声、远处的城市声
- 配乐：悲伤的钢琴曲

高清，4K，画面稳定无抖动，口型同步
```

| 能力 | 说明 |
|------|------|
| **文生图** | 文字描述直接生成高质量图像 |
| **角色图** | 专门优化的角色图片生成 |
| **参考图生图** | 提取参考特征，复刻风格与形象 |

### 使用命令

```bash
# 角色图片
/drama-image --character 25岁职场女性，干练自信 --style cinematic

# 场景图片
/drama-image 城市天台，黄昏 --style cinematic

# 多图输出
/drama-image 生成角色表情变体 --mode multi-output

# 优化提示词
/drama-image --optimize "一个女孩撑伞走在街上"
```

## Seedance 2.0 视频提示词

### 四种模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **标准模式** | 简洁高效的中文描述 | 对话、日常、情感 |
| **融合模式** | 画面+配音+音频一体化 | 对话、独白、情感戏 |
| **时间分段模式** | 15秒内精确控制多个阶段 | 穿越、变身、动作爆发 |
| **单镜头模式** | 一镜到底的长镜头 | 追逐、跑酷、运动场景 |

### 使用命令

```bash
# 标准模式
/drama-prompt 女孩在黄昏街道行走

# 融合模式（含配音）
/drama-prompt 情侣分手场景 --mode fusion

# 时间分段模式
/drama-prompt 女孩穿越到古代 --mode phased

# 批量生成分镜提示词
/drama-prompt --batch 06_分镜脚本/第1集_分镜.md
```

### 核心特性

> **经过实测验证，Seedance 2.0 对中文提示词的支持优于英文！**

- **中文优先**：中文专业术语识别度更高
- **原生配音**：Voice 字段可生成同步口型的配音
- **音画同步**：同时生成匹配的音效、配乐与配音
- **多模态参考**：支持图片、视频、音频混合输入

## 与即梦AI的配合

1. **角色图**：`/drama-image --character` → 即梦AI → 保存到 `03_角色图片/`
2. **场景图**：`/drama-image` → 即梦AI → 保存到 `05_场景图片/`
3. **视频**：复制提示词（含 `@角色图 @场景图`）→ 即梦AI → 一键生成带配音视频
4. **后期**：剪辑软件整合

### 参考图命名规范

```
角色：@角色名_类型.jpg    例：@林小美_半身.jpg
场景：@场景名_视角.jpg    例：@天台_全景_黄昏.jpg
```

## 质量保证环节

### fill-prompts：补充缺失提示词

```bash
/drama-fill-prompts
```

- 扫描所有分镜文件
- 识别缺失/异常的提示词
- 根据分镜描述自动补充
- 生成补充报告

### review：分镜审核

| 维度 | 检查项 |
|------|--------|
| 完整性 | 所有镜头都有提示词 |
| 内容一致性 | 场景、角色、动作、时间、情绪 |
| 技术规范性 | 运镜匹配、时长合理、格式规范 |
| 连贯性 | 镜头衔接、角色状态、场景连续 |
