# AI 短剧文字工作流

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> Claude Code Plugin - 在 Claude Code 中完成 AI 短剧的全部文字创作工作

## 简介

这是一个 Claude Code 插件，用于系统化地完成 AI 短剧的文字创作工作。生成的提示词和台本可直接用于即梦AI等平台进行视频制作。

## 安装

### 方式一：插件市场安装（推荐）

在 Claude Code 中运行以下命令：

```bash
# 添加插件市场
/plugin marketplace add hestudy/ai-short-drama

# 安装插件
/plugin install ai-short-drama@ai-short-drama
```

### 方式二：克隆到用户级目录

```bash
git clone https://github.com/hestudy/ai-short-drama.git ~/.claude/plugins/ai-short-drama
```

### 方式三：项目级安装

```bash
mkdir -p your-project/.claude/plugins
git clone https://github.com/hestudy/ai-short-drama.git your-project/.claude/plugins/ai-short-drama
```

## 使用方法

### 独立命令（推荐）

每个阶段都是独立的斜杠命令，带有清晰的描述：

```
/drama-new "剧名" --episodes 10   # 创建新项目
/drama-plan                       # 策划故事大纲
/drama-character                  # 设计角色
/drama-storyboard --all           # 生成分镜脚本
/drama-fill-prompts               # 补充缺失提示词
/drama-review --fix               # 审核并修正
/drama-script --all               # 生成配音台本
/drama-merge-voice                # ⭐ 合并旧版到融合格式
/drama-export                     # 导出提示词
```

### 工作流概览

输入 `/ai-short-drama` 可查看完整的命令列表和工作流程说明。

### 自然语言触发

直接说：
- "我想创作一个AI短剧"
- "帮我写短剧"
- "生成短剧提示词"

## 工作流程

```

│ drama-new │ → │ drama-plan│ → │drama-character│ → │drama-storyboard│ ← 融合模式
└───────────┘   └───────────┘   └───────────────┘   └───────┬───────┘
                                                        │
      ┌─────────────────────────────────────────────────┘
      ↓
┌─────────────────┐   ┌─────────────┐   ┌──────────────┐
│drama-fill-prompts│ → │drama-review │ → │drama-export  │
└─────────────────┘   └─────────────┘   └──────┬───────┘
```

## 质量保证环节

### fill-prompts：补充缺失提示词

检查并补充分镜中缺失或不完整的AI提示词：

```bash
/drama-fill-prompts
```

**执行内容：**
- 扫描所有分镜文件
- 识别缺失/异常的提示词
- 根据分镜描述自动补充
- 生成补充报告

### review：分镜审核

确保提示词准确还原分镜设计：

**审核维度：**

| 维度 | 检查项 |
|------|--------|
| 完整性 | 所有镜头都有提示词（优先检查） |
| 内容一致性 | 场景、角色、动作、时间、情绪是否一致 |
| 技术规范性 | 运镜匹配、时长合理、格式规范 |
| 连贯性 | 镜头衔接、角色状态、场景连续 |

**审核输出：**
- 逐镜头对比报告
- 问题清单与修正建议
- 修正后的提示词
- 审核通过标记

## 提示词示例

```

```

### Seedance 2.0 融合格式（推荐）⭐

支持画面+配音+音频一体化生成：

```
Subject: [场景核心主体]
Action: [主体动作，越具体越好]
Camera: [镜头尺寸+镜头运动+镜头角度]
Style: [视觉风格]，[灯光效果]

Voice:
- 角色：[说话角色]
- 声音：[声音特点]
- 台词：「[台词内容]」
- 情绪：[情绪描述]
- 语速：[正常/偏快/偏慢]

Audio:
- 环境音：[环境声音]
- 配乐：[背景音乐风格]
- 音效：[特殊音效]

Constraints: 口型同步，[视频时长]
```

### 示例

**融合格式（Seedance 2.0）：**
```
Subject: 25岁职场女性，穿着职业套装，背着公文包
Action: 疲惫地走在黄昏的街道上，偶尔抬头看看天，眼神透露着孤独
Camera: 中景，侧面跟随，缓慢移动

Voice:
- 角色：女主角
- 声音：温柔女声，略带疲惫
- 台词：「又是一天结束了...」
- 情绪：低落/疲惫

Audio:
- 环境音：城市傍晚车流声，远处人声
- 配乐：舒缓忧伤的钢琴曲
- 音效：高跟鞋踩在地面上的声音
```

**通用格式：**
```
黄昏时分的城市街道，夕阳将天空染成橙红色。一位25岁的职场女性，穿着职业套装，疲惫地走在人行道上，偶尔抬头看看天，眼神中透露着孤独。温暖的橙色调，电影感构图，镜头从侧面中景跟随她缓慢移动。
```

## 与即梦AI的配合

1. **在 Claude Code 中**：完成所有文字创作
2. **导出提示词**：复制生成的提示词和台本
3. **在即梦AI中**：
   - 角色外观提示词 → 生成角色参考图
   - 场景提示词 → 生成场景参考图
   - 视频提示词 + 参考图 → 生成视频片段
   - 配音台本 → AI配音
4. **后期整合**：在剪辑软件中完成最终成片

## Seedance 2.0 支持

本插件已集成 Seedance 2.0 提示词最佳实践，支持**融合模式**一键生成带配音的视频！

### 核心能力
| 能力 | 说明 |
|------|------|
| **原生配音** ⭐ | Voice 字段可生成同步口型的配音 |
| **音画同步** | 同时生成匹配的音效、配乐与配音 |
| 自运镜/自分镜 | 模型自动规划镜头运动和分镜切换 |
| 角色一致性 | 多镜头间人物特征保持稳定 |
| 多模态参考 | 支持图片、视频、音频混合输入 |

### 融合模式 vs 传统模式

| 对比项 | 融合模式 | 传统模式 |
|--------|----------|----------|
| 提示词 | 画面+配音+音频一体化 | 仅画面描述 |
| 生成步骤 | 1步完成 | 2步（视频+配音） |
| 效率提升 | 50%+ | 基准 |
| 适用场景 | 有台词的剧情短剧 | 无声画面/后期单独配音 |

### 从旧版迁移

如果你已有使用旧版工作流创建的项目（分镜和配音分离），可以使用迁移命令：

```bash
# 将旧版分镜+配音合并为融合格式
/drama-merge-voice

# 仅迁移指定集数
/drama-merge-voice 1
/drama-merge-voice 1-3
```

迁移后建议：
1. 检查融合提示词的完整性
2. 运行 `/drama-review --fix` 进行一致性检查
3. 运行 `/drama-export` 导出新的融合提示词包

### 多模态参考语法
```
@图片1 作为首帧
@图片2 作为角色外观参考
@视频1 作为镜头运镜参考
@音频1 用于生成配乐
```
