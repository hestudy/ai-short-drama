> Claude Code Plugin - 在 Claude Code 中完成 AI 短剧的全部文字创作工作

## 简介

这是一个 Claude Code 插件，用于系统化地完成 AI 短剧的文字创作工作。生成的提示词和台本可直接用于即梦AI等平台进行视频制作。

**🔥 v2.0.0 重大更新**：
- **中文提示词更优**：经过实测验证，Seedance 2.0 对中文的支持优于英文！
- **质量词强制要求**：每条提示词必须包含质量词（高清、4K、画面稳定、无抖动）
- **融合模式增强**：画面+配音+音频一体化生成
- **四种提示词模式**：标准模式、融合模式、时间分段模式、单镜头模式
- 整合 [awesome-seedance-2-prompts](https://github.com/YouMind-OpenLab/awesome-seedance-2-prompts) 最佳实践

## 安装

### 方式一：插件市场安装（推荐）

在 Claude Code 中运行以下命令：

```bash
# 添加插件市场
/plugin marketplace add hestudy/ai-short-drama

# 安装插件
/plugin install ai-short-drama@ai-short-drama
```

### 方式二：克隆到用户级目录

```bash
git clone https://github.com/hestudy/ai-short-drama.git ~/.claude/plugins/ai-short-drama
```

### 方式三：项目级安装

```bash
mkdir -p your-project/.claude/plugins
git clone https://github.com/hestudy/ai-short-drama.git your-project/.claude/plugins/ai-short-drama
```

## 使用方法

### 独立命令（推荐）

每个阶段都是独立的斜杠命令，带有清晰的描述：

```
/drama-new "剧名" --episodes 10   # 创建新项目
/drama-plan                       # 策划故事大纲
/drama-character                  # 设计角色
/drama-storyboard --all           # 生成分镜脚本
/drama-fill-prompts               # 补充缺失提示词
/drama-review --fix               # 审核并修正
/drama-script --all               # 生成配音台本
/drama-merge-voice                # ⭐ 合并旧版到融合格式
/drama-prompt [场景] --mode fusion # ⭐ Seedance 2.0 提示词专家
/drama-export                     # 导出提示词
```

## 质量保证环节

### fill-prompts：补充缺失提示词

检查并补充分镜中缺失或不完整的AI提示词：

```bash
/drama-fill-prompts
```

**执行内容：**
- 扫描所有分镜文件
- 识别缺失/异常的提示词
- 根据分镜描述自动补充
- 生成补充报告

### review：分镜审核

确保提示词准确还原分镜设计：

**审核维度：**

| 维度 | 检查项 |
|------|--------|
| 完整性 | 所有镜头都有提示词（优先检查） |
| 内容一致性 | 场景、角色、动作、时间、情绪是否一致 |
| 技术规范性 | 运镜匹配、时长合理、格式规范 |
| 连贯性 | 镜头衔接、角色状态、场景连续 |

**审核输出：**
- 逐镜头对比报告
- 问题清单与修正建议
- 修正后的提示词
- 审核通过标记

## 提示词示例

> **核心发现**：Seedance 2.0 对中文提示词的支持**优于英文**！
> - 使用中文专业术语（俯拍、跟拍、环绕拍）效果更精准
> - 每条提示词必须包含质量词：高清、4K、画面稳定、无抖动

### 提示词模式 🆕

本插件支持四种提示词模式：

| 模式 | 说明 | 适用场景 | 时长建议 |
|------|------|----------|----------|
| **标准模式** | 简洁高效的中文描述 | 对话、日常、情感 | 4-6秒 |
| **融合模式** | 画面+配音+音频一体化 | 对话、独白、情感戏 | 4-15秒 |
| **时间分段模式** | 15秒内精确控制多个阶段 | 穿越、变身、动作爆发 | 10-15秒 |
| **单镜头模式** | 一镜到底的长镜头 | 追逐、跑酷、运动场景 | 10-15秒 |

### 使用 /drama-prompt 命令

```bash
# 标准模式（默认）
/drama-prompt 女孩在黄昏街道行走

# 融合模式（含配音）
/drama-prompt 情侣分手场景 --mode fusion

# 时间分段模式
/drama-prompt 女孩穿越到古代 --mode phased

# 单镜头模式
/drama-prompt 摩托车穿越村庄追逐 --mode continuous

# 优化现有提示词
/drama-prompt --optimize "一个人在跑步"
```

### 质量词（必加）⭐

**每条提示词必须包含**：
```
高清，4K，画面稳定无抖动，细节丰富，面部清晰，无变形
```

## 与即梦AI的配合

1. **在 Claude Code 中**：完成所有文字创作
2. **导出提示词**：复制生成的提示词和台本
3. **在即梦AI中**：
   - 角色外观提示词 → 生成角色参考图
   - 场景提示词 → 生成场景参考图
   - 视频提示词 + 参考图 → 生成视频片段
   - 配音台本 → AI配音
4. **后期整合**：在剪辑软件中完成最终成片

## Seedance 2.0 支持

本插件已集成 Seedance 2.0 提示词最佳实践，支持**融合模式**一键生成带配音的视频！

### 核心发现 🔥

> **经过实测验证，Seedance 2.0 对中文提示词的支持优于英文！**
> - 中文专业术语（俯拍、跟拍、环绕拍）识别度和精准度更高
> - 建议优先使用中文描述场景、动作和镜头

### 核心能力
| 能力 | 说明 |
|------|------|
| **中文优先** ⭐ | 对中文提示词的支持优于英文 |
| **原生配音** ⭐ | Voice 字段可生成同步口型的配音 |
| **音画同步** | 同时生成匹配的音效、配乐与配音 |
| 自运镜/自分镜 | 模型自动规划镜头运动和分镜切换 |
| 角色一致性 | 多镜头间人物特征保持稳定 |
| 多模态参考 | 支持图片、视频、音频混合输入 |

### 融合模式 vs 传统模式

| 对比项 | 融合模式 | 传统模式 |
|--------|----------|----------|
| 提示词 | 画面+配音+音频一体化 | 仅画面描述 |
| 生成步骤 | 1步完成 | 2步（视频+配音） |
| 效率提升 | 50%+ | 基准 |
| 适用场景 | 有台词的剧情短剧 | 无声画面/后期单独配音 |

### 从旧版迁移

如果你已有使用旧版工作流创建的项目（分镜和配音分离），可以使用迁移命令：

```bash
# 将旧版分镜+配音合并为融合格式
/drama-merge-voice

# 仅迁移指定集数
/drama-merge-voice 1
/drama-merge-voice 1-3
```

迁移后建议：
1. 检查融合提示词的完整性
2. 运行 `/drama-review --fix` 进行一致性检查
3. 运行 `/drama-export` 导出新的融合提示词包
