# AI 短剧文字工作流

> Claude Code Skill - 在 Claude Code 中完成 AI 短剧的全部文字创作工作

## 简介

这是一个 Claude Code 插件（Skill），用于系统化地完成 AI 短剧的文字创作工作。生成的提示词和台本可直接用于即梦AI等平台进行视频制作。

## 功能特性

- 📝 **完整工作流**：从故事策划到提示词导出的全流程支持
- 🎬 **即梦AI适配**：提示词专为即梦AI优化，可直接复制使用
- 📊 **结构化输出**：标准化的文档结构，方便管理和协作
- 🔄 **可复用模板**：提供丰富的提示词模板库

## 安装

### 方式一：克隆到 Claude Code Skills 目录

```bash
cd ~/.claude/skills/
git clone https://github.com/hestudy/ai-short-drama.git
```

### 方式二：作为项目级 Skill

在项目根目录创建 `.claude/skills/` 目录，然后克隆：

```bash
mkdir -p your-project/.claude/skills
cd your-project/.claude/skills
git clone https://github.com/hestudy/ai-short-drama.git
```

## 使用方法

### 触发方式

在 Claude Code 中，通过以下方式触发：

```
我想创作一个AI短剧
帮我写短剧
AI短剧创作
```

### 工作流程

```
阶段一：策划 → 阶段二：角色设计 → 阶段三：分镜脚本 → 阶段四：配音台本 → 阶段五：导出
```

### 输出物

| 输出物 | 说明 | 用途 |
|--------|------|------|
| 故事大纲 | 完整的故事框架 | 规划参考 |
| 角色设定卡 | 角色信息 + 外观提示词 | 即梦AI角色图生成 |
| 分镜脚本 | 详细的镜头拆解 | 制作指导 |
| 视频提示词 | 每个镜头的即梦AI提示词 | 即梦AI视频生成 |
| 配音台本 | 角色台词 + 情绪标注 | 即梦AI配音 |
| 制作清单 | 制作进度跟踪 | 项目管理 |

## 与即梦AI的配合

1. **在 Claude Code 中**：完成所有文字创作（策划、剧本、提示词）
2. **导出提示词**：复制生成的提示词和台本
3. **在即梦AI中**：
   - 角色外观提示词 → 生成角色参考图
   - 场景提示词 → 生成场景参考图
   - 视频提示词 + 参考图 → 生成视频片段
   - 配音台本 → AI配音
4. **后期整合**：在剪辑软件中完成最终成片

## 示例

```
用户：我想创作一个10集的都市情感AI短剧，主题是"社恐女孩的恋爱逆袭"

Claude：好的，我来帮你开始这个AI短剧的创作。首先进入策划阶段...

[按照工作流逐步完成各阶段]

最终输出：
- 10集完整分镜脚本
- 所有角色的即梦AI提示词
- 所有镜头的视频生成提示词
- 完整配音台本
```

## 提示词模板示例

### 视频生成提示词结构
```
[场景描述] + [主体特征与动作] + [视觉风格] + [情绪或氛围] + [镜头语言]
```

### 示例
```
黄昏时分的城市街道，夕阳将天空染成橙红色。一位25岁的职场女性，穿着职业套装，疲惫地走在人行道上，偶尔抬头看看天，眼神中透露着孤独。温暖的橙色调，电影感构图，镜头从侧面中景跟随她缓慢移动。
```
