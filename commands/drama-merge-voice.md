# /drama-merge-voice - 合并配音到融合格式

将旧版分离的分镜提示词和配音台本合并为 Seedance 2.0 融合格式。

## 用法

```bash
/drama-merge-voice [集数]
```

**参数：**
- `集数`：可选，指定要合并的集数（如 `1` 或 `1-3`），不指定则处理所有集数

## 前置条件

- 项目已有 `03_分镜脚本/` 和 `05_配音台本/` 目录
- 分镜脚本包含视频提示词
- 配音台本包含角色台词和情绪标注

## 执行流程

### Step 1: 扫描现有文件

扫描项目目录结构：
```
03_分镜脚本/
├── 第1集_分镜.md
├── 第2集_分镜.md
└── ...

05_配音台本/
├── 第1集_配音.md
├── 第2集_配音.md
└── ...
```

### Step 2: 匹配分镜与配音

对每个镜头：
1. 从分镜脚本提取镜头编号和视频提示词
2. 从配音台本匹配对应镜头的配音信息
3. 合并为融合格式

### Step 3: 生成融合提示词

将匹配的信息按融合格式重组：

**输入（分镜）：**
```
S01_C01
场景：黄昏街道
提示词：黄昏时分的城市街道...
```

**输入（配音）：**
```
S01_C01
角色：女主角
台词：「又是一天结束了」
情绪：疲惫
语速：偏慢
```

**输出（融合格式）：**
```
Subject: 25岁职场女性，穿着职业套装
Action: 疲惫地走在黄昏的街道上...
Camera: 中景，侧面跟随
Style: 电影感，温暖橙色调

Voice:
- 角色：女主角
- 声音：温柔女声，略带疲惫
- 台词：「又是一天结束了」
- 情绪：疲惫
- 语速：偏慢

Audio:
- 环境音：城市傍晚车流声
- 配乐：舒缓忧伤的钢琴曲
- 音效：高跟鞋踩在地面上的声音

Constraints: 口型同步，6秒
```

### Step 4: 输出融合文件

创建新的融合提示词文件：

```
04_视频提示词/
├── 第1集/
│   ├── 融合提示词.md    ← 新生成的融合格式
│   └── ...
└── ...
```

### Step 5: 生成迁移报告

输出迁移报告，包含：
- 处理的集数和镜头数
- 成功匹配的数量
- 无法匹配的镜头（需人工检查）
- 音频字段建议（基于分镜描述推断）

## 迁移报告格式

```markdown
# 融合格式迁移报告

## 处理概要
| 集数 | 镜头数 | 成功匹配 | 无配音 | 需检查 |
|------|--------|----------|--------|--------|
| 第1集 | 15 | 12 | 3 | 0 |
| 第2集 | 18 | 15 | 2 | 1 |

## 无法匹配的镜头
- 第2集 S02_C08：配音台本中无对应记录

## 音频字段推断
基于分镜描述自动推断的音频信息：
- S01_C01：推断环境音为「城市街道」
- S01_C05：推断配乐为「悲伤氛围」

## 下一步建议
1. 检查「无法匹配的镜头」
2. 确认推断的音频字段是否正确
3. 运行 `/drama-review` 进行一致性检查
4. 运行 `/drama-export` 重新导出融合提示词包
```

## 注意事项

1. **保留原文件**：原分镜和配音文件不会被修改
2. **音频推断**：Audio 字段基于分镜描述推断，需人工确认
3. **无配音镜头**：无台词的镜头仅保留画面信息
4. **声音特点**：需在角色设定中补充，或在迁移时手动指定

## 示例

```bash
# 合并所有集数
/drama-merge-voice

# 仅合并第1集
/drama-merge-voice 1

# 合并第1-3集
/drama-merge-voice 1-3
```

## 执行后

迁移完成后，建议：
1. 检查融合提示词的完整性
2. 补充缺失的音频信息
3. 运行 `/drama-review --fix` 进行一致性检查
4. 运行 `/drama-export` 导出新的融合提示词包
