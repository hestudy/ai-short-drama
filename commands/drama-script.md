# 配音台本

生成配音文本，包含台词、情绪标注和语速建议。

## 用法

```
/drama-script              # 交互式选择集数
/drama-script --all        # 生成所有集的配音台本
/drama-script 3            # 生成第3集配音台本
/drama-script 1-5          # 生成第1-5集配音台本
```

## 前置条件

- 已完成分镜脚本创作
- 建议先通过审核确保质量

## 执行流程

> ⚠️ **并发控制**：批量处理，每批 20 条台词，批次间隔 1000ms

1. 确认要处理的集数
2. 读取分镜脚本
3. 提取对话内容
4. 批量添加配音指导（每批 20 条台词）：
   - 情绪标注
   - 语速建议
   - 停顿标记
5. 添加旁白和内心独白
6. 生成配音台本文件

## 输出文件

```
05_配音台本/
└── 第X集_配音.md
```

## 台本格式

```markdown
# 第1集 配音台本

## 基本信息
- 集数：第1集
- 总时长：约3分钟
- 主要角色：林晓月、陈默

## 配音内容

### S01_C01 场景1-街道-日
**[环境音：城市街道，傍晚车流]**

> **旁白**（沉稳、略带沧桑）：
> 在这座城市里，每个人都在奔波。
>
> [停顿 1秒]

---

### S01_C02 场景2-办公室-日

**林晓月**（疲惫、无奈）：
> 又是加班的一天...
>
> [语速：慢 | 情绪：低落]

---

### S01_C05 场景5-电梯-日
**陈默**（礼貌、带笑意）：
> 晚上好，林小姐。加班到现在？
>
> [语速：正常 | 情绪：温和]

**林晓月**（惊讶、略紧张）：
> 啊...是的，陈总。您也还没走？
>
> [语速：稍快 | 情绪：紧张]
```

## 情绪标注库

| 标签 | 说明 | 适用场景 |
|------|------|----------|
| 平静 | 正常语气 | 日常对话 |
| 激动 | 高昂、快速 | 争吵、兴奋 |
| 低落 | 低沉、缓慢 | 悲伤、沮丧 |
| 紧张 | 急促、不稳 | 危机、焦虑 |
| 温柔 | 轻柔、温暖 | 关心、爱意 |
| 愤怒 | 有力、急促 | 冲突、爆发 |

## 示例

```
/drama-script              # 交互式
/drama-script --all        # 全部集数
/drama-script 1            # 只做第1集
```
