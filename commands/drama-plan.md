# 策划阶段

生成故事大纲，确定核心概念和剧情走向。

## 用法

```
/drama-plan
```

## 前置条件

- 已运行 `/drama-new` 创建项目
- 或当前目录存在 `00_项目配置.md`

## 执行流程

1. 读取项目配置，确认基本信息
2. 收集策划需求：
   - 故事类型与题材
   - 核心卖点/钩子
   - 目标受众
   - 视觉风格偏好
3. 生成故事大纲：
   - 一句话故事核心
   - 三幕结构
   - 分集大纲
4. 确定视觉风格参考
5. 写入 `01_故事大纲.md`

## 输出内容

```markdown
# 故事大纲

## 核心信息
- 剧名：
- 类型：
- 集数：
- 核心卖点：

## 一句话故事
[50字以内概括全剧]

## 三幕结构
### 第一幕：铺垫
### 第二幕：冲突
### 第三幕：高潮与结局

## 分集大纲
[每集50-100字概述]

## 视觉风格
[风格关键词、色调参考]
```

## 示例

```
/drama-plan
```
