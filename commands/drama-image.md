# Seedream 4.0-4.5 图片提示词专家

为火山方舟 Seedream 4.0-4.5 生成高质量图片提示词的专业工具。

> **核心优势**：Seedream 4.0/4.5 对中文提示词支持优秀，简洁精准的描述远优于堆砌复杂词汇。

## 用法

```bash
# 基础用法 - 文生图
/drama-image [场景描述]

# 指定模式
/drama-image [描述] --mode [text2img|img2img|ref2img|multi-input|multi-output]

# 指定风格
/drama-image [描述] --style [realistic|cinematic|anime|cyberpunk|vintage|ancient|oil-painting]

# 角色图片
/drama-image --character [角色描述]

# 优化现有提示词
/drama-image --optimize [现有提示词]

# 图片编辑
/drama-image --edit [编辑指令]
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `[描述]` | 简要描述你想要生成的图像 | - |
| `--mode` | 提示词模式 | text2img |
| `--style` | 视觉风格 | realistic |
| `--character` | 生成角色外观提示词 | - |
| `--optimize` | 优化现有提示词 | - |
| `--edit` | 图片编辑（增加/删除/替换/修改） | - |
| `--version` | 模型版本 | 4.5 |

### 模式说明

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| `text2img` | 文生图，文字直转图像 | 角色设计、场景创作、知识图表 |
| `img2img` | 图生图，图文结合编辑 | 局部修改、增加删除、替换修改 |
| `ref2img` | 参考图生图，提取特征复刻 | 角色换装、风格迁移、草图转效果图 |
| `multi-input` | 多图输入，复合编辑 | 跨图替换、组合、风格迁移 |
| `multi-output` | 多图输出，系列组图 | 品牌VI、分镜、表情包、壁纸 |

### 风格说明

| 风格 | 说明 | 关键词 |
|------|------|--------|
| `realistic` | 写实主义，自然光线 | 自然光线、写实色彩、高细节 |
| `cinematic` | 电影质感，戏剧光影 | 电影感、高对比、戏剧性用光 |
| `anime` | 动漫风格，鲜艳色彩 | 日漫风、鲜艳色彩、夸张表现 |
| `cyberpunk` | 赛博朋克，霓虹灯光 | 霓虹灯光、蓝紫色调、未来感 |
| `vintage` | 复古怀旧，胶片感 | 暖黄色调、胶片颗粒感、柔和边缘 |
| `ancient` | 古风唯美，水墨意境 | 水墨、古典美学、暖色调、飘逸感 |
| `oil-painting` | 油画风格，莫奈印象派 | 莫奈风格、印象派、油画质感 |
| `fresh` | 日系清新，柔和光线 | 柔和光线、高调色彩、浅景深、梦幻氛围 |

## 执行流程

### Step 1: 分析需求

1. 解析用户输入的描述
2. 确定提示词模式（默认 text2img）
3. 确定视觉风格（默认 realistic）
4. 确定是否为角色图片

### Step 2: 查阅参考资源

查阅 `skills/seedream-prompts/references/` 目录下的参考文档：
- `prompt-guidelines.md` - 提示词编写规范
- `prompt-templates.md` - 提示词模板库

### Step 3: 生成提示词

**必须遵循的五大通用规则**：

1. **自然语言清晰描述**：用连贯语句写明主体+行为+环境
2. **明确应用场景**：直接标明图像类型和用途
3. **精准风格词**：使用具体风格关键词或参考图
4. **文字加引号**：需要渲染的文字用双引号包裹
5. **清晰编辑指令**：说明修改对象和具体操作，避免模糊代词

---

**文生图模板**：
```
[场景描述]，[时间/天气/光线]。
[主体描述]，[具体动作描述]，[情绪/状态]。
[视觉风格]，[色调/光影]。
```

**角色图片模板**：
```
一位[年龄]的[性别]，[发型发色]，[面部特征]，
穿着[服装描述]，[配饰描述]，
[表情]，[姿态/动作]，
[风格]风格，[光影]光影。
```

**图生图模板（编辑）**：
```
[编辑操作]图中的[对象]，[具体描述]，保持[保持要求]
```

**参考图生图模板**：
```
参考图中的人物形象/艺术风格/产品款式，[生成描述]
```

**多图输出模板**：
```
生成[数量]张[内容描述]，每张图[具体要求]，风格统一
```

### Step 4: 质量检查

**检查清单**：
- [ ] 使用连贯的自然语言描述
- [ ] 明确主体、行为、环境
- [ ] 有具体应用场景时已标明
- [ ] 需要渲染的文字已用引号包裹
- [ ] 编辑操作明确具体，无模糊代词
- [ ] 风格关键词具体明确

### Step 5: 输出结果

生成可直接复制到即梦AI使用的提示词。

## 示例

### 示例1：文生图 - 场景创作

```
输入：
/drama-image 女孩在黄昏的街道上行走，表情孤独

输出：
黄昏时分的城市街道，夕阳将天空染成橙红色。
一位25岁的职场女性，穿着职业套装，背着公文包，
疲惫地走在人行道上，偶尔抬头看看天，眼神中透露着孤独。
电影感构图，温暖的橙色调，柔和光线。
```

### 示例2：文生图 - 办公场景

```
输入：
/drama-image 凌乱的程序员办公桌 --style realistic

输出：
一个凌乱的办公桌桌面，桌上有开着的笔记本电脑（屏幕显示绿色代码）、
印着"Developer"的热乎马克杯、摊开的维恩图书籍（灰蓝绿三色嵌套圆）、
三层结构思维导图便签、笔帽分离的钢笔、显示新消息的手机、角落小多肉，
背景是模糊书架，右侧阳光形成光影。
自然光线，写实色彩，高细节。
```

### 示例3：角色图片

```
输入：
/drama-image --character 25岁职场女性，干练自信

输出：
一位25岁的女性，黑色长发扎成低马尾，瓜子脸，大眼睛，
穿着白色职业衬衫和黑色西装外套，戴着细框眼镜，
表情严肃而自信，双手抱胸站立。
日系职场风格，柔和侧光，高细节，面部清晰。
```

### 示例4：图生图 - 编辑操作

```
输入：
/drama-image --edit 给女孩增加耳环和项链

输出：
给图中女生增加同款银色耳线和项链，保持原有发型和服装不变。
自然光线，写实风格，高细节。
```

### 示例5：参考图生图

```
输入：
/drama-image 参考图中的人物，生成羊毛毡手办 --mode ref2img

输出：
将图中的人物形象转化为羊毛毡材质，搭配小支架维持原有姿势，
放置在深色木质书桌上，室内柔和光线，微距摄影效果，
保持人物的特征表情和姿势，高细节。
```

### 示例6：多图输出

```
输入：
/drama-image 生成周一到周日的自然景观壁纸 --mode multi-output

输出：
生成周一到周日七张自然景观手机壁纸，每张图清晰标注对应日期：
周一：日出山海，金光万道
周二：草原花海，蝴蝶飞舞
周三：森林溪流，阳光斑驳
周四：湖泊倒影，白云悠悠
周五：沙漠日落，橙红漫天
周六：星空银河，璀璨无垠
周日：雪山云海，宁静致远
风格统一，日系清新，柔和光线，9:16竖屏比例。
```

### 示例7：优化提示词

```
输入：
/drama-image --optimize "一个女孩撑伞走在街上"

输出：
黄昏时分的林荫街道，阳光透过树叶洒下斑驳光影。
一位身着淡蓝色连衣裙的女孩，撑着白色蕾丝遮阳伞，
缓缓漫步在街道上，偶尔抬头望向远方，表情温柔恬静。
日系清新风格，柔和光线，浅景深虚化背景，梦幻氛围。

优化建议：
1. 增加具体的时间和天气描述（黄昏、阳光）
2. 增加场景细节（林荫街道、光影）
3. 增加人物特征（连衣裙颜色、遮阳伞样式）
4. 明确动作细节（缓缓漫步、抬头望向）
5. 添加情绪描述（温柔恬静）
6. 确定视觉风格（日系清新）
```
