# Seedance 2.0 提示词专家

为 Seedance 2.0 生成高质量视频提示词的专业工具。

> **核心优势**：Seedance 2.0 对中文提示词的支持优于英文，本工具优先使用中文专业术语。

## 用法

```bash
# 基础用法
/drama-prompt [场景描述]

# 指定模式
/drama-prompt [场景描述] --mode [standard|phased|continuous|fusion]

# 指定风格
/drama-prompt [场景描述] --style [realistic|cinematic|anime|fantasy|scifi|noir]

# 优化现有提示词
/drama-prompt --optimize [现有提示词]

# 批量生成分镜提示词
/drama-prompt --batch [分镜文件路径]
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `[场景描述]` | 简要描述你想要生成的场景 | - |
| `--mode` | 提示词模式 | standard |
| `--style` | 视觉风格 | realistic |
| `--optimize` | 优化现有提示词 | - |
| `--duration` | 视频时长（秒） | 6 |
| `--voice` | 是否包含配音（融合模式） | false |
| `--batch` | 批量生成分镜提示词 | - |

### 模式说明

| 模式 | 说明 | 适用场景 | 时长建议 |
|------|------|----------|----------|
| `standard` | 标准模式，简洁高效 | 对话、日常、情感 | 4-6秒 |
| `phased` | 时间分段模式，精确控制 | 穿越、变身、动作爆发 | 10-15秒 |
| `continuous` | 单镜头模式，一镜到底 | 追逐、跑酷、长镜头 | 10-15秒 |
| `fusion` | 融合模式，画面+配音+音频 | 对话、独白、情感戏 | 4-15秒 |

### 风格说明

| 风格 | 说明 | 关键词 |
|------|------|--------|
| `realistic` | 现实主义，自然光线 | 自然光线、写实色彩、生活化 |
| `cinematic` | 电影质感，戏剧光影 | 电影感、高对比、戏剧性用光 |
| `anime` | 动漫风格，鲜艳色彩 | 日漫风、鲜艳色彩、夸张表现 |
| `fantasy` | 奇幻风格，梦幻光影 | 梦幻、光效、史诗、粒子效果 |
| `scifi` | 科幻风格，霓虹灯光 | 赛博朋克、霓虹、未来感 |
| `noir` | 黑色电影，高对比 | 高对比、阴影深沉、冷色调 |
| `wuxia` | 武侠风格，水墨意境 | 水墨、古典、飘逸、剑气 |
| `ancient` | 古装风格，古典美学 | 古典、暖色调、柔和光线 |

## 执行流程

### Step 1: 分析需求

1. 解析用户输入的场景描述
2. 确定提示词模式（默认 standard）
3. 确定视觉风格（默认 realistic）
4. 确定是否需要配音

### Step 2: 查阅参考资源

查阅 `skills/seedance-prompts/references/` 目录下的参考文档：
- `prompt-templates.md` - 提示词模板库
- `camera-movements.md` - 运镜参考手册
- `scene-types.md` - 场景类型详解
- `quality-keywords.md` - 质量关键词库

### Step 3: 生成提示词

**必须遵循的核心原则**：
1. 优先使用中文描述（Seedance 2.0 对中文支持更优）
2. 每条提示词必须包含质量词
3. 动作描述要具体、连续、可执行
4. 镜头语言要使用中文专业术语

**标准模式模板**：
```
[场景描述]，[时间/天气/光线]。
[主体描述]，[具体动作描述]，[情绪/状态]。
[视觉风格]，[色调/光影]。
[镜头类型]，[运镜描述]。
[质量词]
```

**融合模式模板**（含配音）：
```
[场景描述]，[时间/天气/光线]。
[主体描述]，[具体动作描述]，[情绪/状态]。
[视觉风格]，[色调/光影]。
[镜头类型]，[运镜描述]。

配音：
- 角色：[说话角色]
- 声音：[声音特点]
- 台词：「[台词内容]」
- 情绪：[情绪描述]
- 语速：[语速]

音频：
- 环境音：[背景声音]
- 配乐：[音乐风格]

[质量词]
```

**时间分段模式模板**：
```
[总时长]，[整体风格]

[时间段1]：[画面描述] + [动作描述] + [镜头描述]
[时间段2]：[画面描述] + [动作描述] + [镜头描述]
[时间段3]：[画面描述] + [动作描述] + [镜头描述]

音效：[音频描述]
限制：[负面提示]
[质量词]
```

**单镜头模式模板**：
```
单镜头连续拍摄，[主体描述]
[动作轨迹描述]
镜头[运镜描述]
最终揭示[画面揭示]
[质量词]
```

### Step 4: 质量检查

**必须包含的质量词**（每条提示词必加）：
- 画质类：高清、4K、画面稳定、无抖动
- 细节类：细节丰富、面部清晰、无变形
- 色彩类：色彩自然、无偏色

**检查清单**：
- [ ] 包含明确的主体
- [ ] 动作描述具体连续
- [ ] 镜头类型和运动清晰
- [ ] 风格和氛围一致
- [ ] 时长合理
- [ ] 质量词完整
- [ ] 负面提示（如需要）

### Step 5: 输出结果

生成可直接复制到即梦AI使用的提示词。

## 示例

### 示例1：标准模式

```
输入：
/drama-prompt 女孩在黄昏的街道上行走，表情孤独

输出：
黄昏时分的城市街道，夕阳将天空染成橙红色。
一位25岁的职场女性，穿着职业套装，背着公文包，
疲惫地走在人行道上，偶尔抬头看看天，眼神中透露着孤独。
电影感构图，温暖的橙色调，柔和光线。
中景，侧面跟随，缓慢移动。
高清，4K，画面稳定无抖动，面部清晰不变形
```

### 示例2：时间分段模式

```
输入：
/drama-prompt 女孩穿越到古代 --mode phased

输出：
15秒，中国古装短剧风格，电影画质，暖金色与冷青色交替

0-3秒：现代都市夜景，一位身穿白色连衣裙的年轻女孩站在天桥上低头看手机，突然脚下一滑
镜头：中景，平视，轻微晃动

3-8秒：女孩失重坠落，身体在空中旋转，周围景物扭曲变形，光线拖尾
镜头：环绕跟随，快速旋转，光影流动

8-15秒：画面骤然稳定，女孩落在古代宫殿的花园中，周围樱花飞舞，她惊讶地站起身，低头看自己的衣服已变成古装
镜头：仰拍全景推进至中景，樱花特写

音效：都市车流声 → 坠落风声 → 古琴轻响
限制：无模糊，人物面部清晰
高清，4K，画面稳定，细节丰富
```

### 示例3：融合模式（含配音）

```
输入：
/drama-prompt 情侣分手场景，女方说出决绝的话 --mode fusion

输出：
深夜的街道，路灯投下冷白的光，细雨纷纷。
一位25岁的年轻女性，穿着单薄的外套，站在雨中，
眼眶含泪，猛然转身面对对方，双手握拳，身体微微颤抖。
电影感构图，冷色调，雨丝在灯光下闪烁。
近景固定，突然推向面部特写。

配音：
- 角色：女主角
- 声音：激动颤抖的女声
- 台词：「我们结束了。」
- 情绪：愤怒/悲伤/决绝
- 语速：缓慢，逐字说出

音频：
- 环境音：雨声、远处车流
- 配乐：悲伤的钢琴曲

高清，4K，画面稳定无抖动，口型同步，表情同步，面部清晰
```

### 示例4：单镜头模式

```
输入：
/drama-prompt 摩托车穿越地中海村庄追逐 --mode continuous

输出：
单镜头连续拍摄，摩托车在地中海风格村庄的狭窄街道中疾驰
镜头穿梭于晾晒的衣物和市场摊位之间，骑手惊险躲避障碍物
镜头跟随环绕，穿越窗户和门廊
最终揭示画面打开至悬崖边缘，俯瞰黄金时刻的无限海洋
高清，4K，画面稳定，动作流畅，细节丰富
```

### 示例5：优化提示词

```
输入：
/drama-prompt --optimize "一个人在跑步"

输出：
公园跑道上，午后阳光透过树叶洒下斑驳光影。
一位30岁左右的男性跑者，穿着蓝色运动背心和黑色短裤，额头有汗珠，
中等速度奔跑，呼吸有节奏，偶尔抬手擦汗，步伐稳健。
健康活力的氛围，自然光线，写实风格。
侧面中景跟随，保持稳定距离。
高清，4K，画面稳定无抖动，动作流畅，细节丰富

优化建议：
1. 增加具体的场景描述（地点、时间、光线）
2. 增加人物细节（年龄、穿着、状态）
3. 明确跑步的速度和动作细节
4. 添加运镜描述
5. 确定风格和氛围
6. 添加必要的质量词
```

## 中文专业术语参考

### 镜头术语
| 中文 | 效果 | 适用场景 |
|------|------|----------|
| 俯拍 | 从上往下拍摄 | 展示全貌、渺小感 |
| 仰拍 | 从下往上拍摄 | 崇高感、压迫感 |
| 跟拍 | 镜头跟随主体移动 | 行走、追逐 |
| 环绕拍 | 镜头围绕主体旋转 | 展示人物、英雄时刻 |
| 推镜 | 镜头向前推进 | 强调、聚焦、紧张 |
| 拉镜 | 镜头向后拉远 | 揭示、展示、结尾 |
| 固定镜头 | 镜头不动 | 沉稳、观察、对话 |

### 景别术语
| 中文 | 描述 |
|------|------|
| 大远景 | 展示环境全貌 |
| 远景 | 展示场景与人物关系 |
| 全景 | 人物全身 |
| 中景 | 人物膝盖以上 |
| 近景 | 人物胸部以上 |
| 特写 | 人物面部 |
| 大特写 | 眼睛、嘴唇等局部 |
