# 分镜审核

审核分镜与提示词的一致性，输出审核报告和修正建议。

## 用法

```
/drama-review
/drama-review --fix         # 审核并自动修正
/drama-review --report      # 只生成报告，不修改文件
```

## 前置条件

- 已运行 `/drama-storyboard` 生成分镜脚本
- 建议先运行 `/drama-fill-prompts` 确保提示词完整

## 审核维度

### 1. 完整性检查（优先）
- 所有镜头都有提示词
- 提示词长度合理（>50字）

### 2. 内容一致性
- 场景匹配：分镜场景与提示词描述一致
- 角色匹配：出场人物与描述一致
- 动作匹配：分镜动作与提示词动作对应
- 情绪匹配：标注情绪与提示词氛围一致
- 时间匹配：日/夜/黄昏等时间设定

### 3. 技术规范性
- 运镜方式合理（推/拉/摇/移/跟）
- 时长合理（2-8秒）
- 格式规范

### 4. 连贯性
- 镜头衔接自然
- 角色状态连续
- 场景转换合理

## 执行流程

> ⚠️ **并发控制**：批量审核，每批 10 个镜头；修正时每批 5 个镜头

1. 读取所有分镜文件
2. 批量进行四维审核（每批 10 个镜头）
3. 标记不一致项
4. 生成审核报告
5. 如使用 `--fix`，批量自动修正不一致项（每批 5 个）

## 输出示例

```markdown
# 分镜审核报告

## 审核概览
- 审核文件：3 集
- 总镜头数：45 个
- 通过镜头：38 个
- 问题镜头：7 个

## 问题详情

### 第1集
| 镜头 | 问题类型 | 问题描述 | 修正建议 |
|------|----------|----------|----------|
| S01_C03 | 场景不一致 | 分镜为"室内"，提示词为"街道" | 修正提示词场景描述 |
| S01_C08 | 情绪不匹配 | 分镜标注"愤怒"，提示词氛围"平静" | 调整情绪描述 |

### 第2集
| 镜头 | 问题类型 | 问题描述 | 修正建议 |
|------|----------|----------|----------|
| S02_C05 | 时长异常 | 标注10秒，超过建议上限 | 拆分为两个镜头 |

## 审核结论
- 整体质量：良好
- 需要修正：7 处
- 建议操作：使用 /drama-review --fix 自动修正
```

## 示例

```
/drama-review              # 审核并显示报告
/drama-review --fix        # 审核并自动修正
/drama-review --report     # 只生成报告
```
