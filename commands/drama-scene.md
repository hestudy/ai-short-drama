# 场景设计

为AI短剧创建专业的场景设定卡和图片提示词。

## 用法

```bash
# 基础用法 - 创建单个场景
/drama-scene [场景名]

# 指定场景类型
/drama-scene [场景名] --type [indoor|outdoor|fantasy]

# 指定时段
/drama-scene [场景名] --time [day|dusk|night]

# 从故事大纲提取所有场景
/drama-scene --all

# 批量生成场景图片提示词
/drama-scene --batch-prompts
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `[场景名]` | 场景名称 | - |
| `--type` | 场景类型 | indoor |
| `--time` | 默认时段 | day |
| `--all` | 从故事大纲提取所有场景 | - |
| `--batch-prompts` | 批量生成图片提示词 | - |

### 场景类型说明

| 类型 | 说明 | 示例 |
|------|------|------|
| `indoor` | 室内场景 | 咖啡馆、办公室、卧室 |
| `outdoor` | 室外场景 | 街道、公园、天台 |
| `fantasy` | 虚构场景 | 仙界、异世界、未来城市 |

### 时段说明

| 时段 | 光线特点 | 情绪氛围 |
|------|----------|----------|
| `day` | 自然光，明亮 | 明亮、活力 |
| `dusk` | 夕阳，暖色调 | 温暖、浪漫、孤独 |
| `night` | 人工光，暗调 | 神秘、压抑、宁静 |

## 执行流程

### Step 1: 确认项目状态

1. 检查是否存在 `00_项目配置.md`
2. 检查是否存在 `01_故事大纲.md`
3. 确认场景存放目录 `05_场景设定/`

### Step 2: 场景识别

**如果使用 `--all` 参数**：
1. 读取 `01_故事大纲.md`
2. 自动识别所有场景
3. 生成场景清单供确认

**如果指定场景名**：
1. 直接进入设定创建流程

### Step 3: 创建场景设定卡

使用 `references/scene-card.md` 模板创建设定卡：

```markdown
# 场景设定卡：[场景名]

## 基本信息
...

## 空间描述
...

## 氛围参数
...

## 环境动态
...

## 时段变体
...

## 一致性控制
...
```

**输出路径**：`[项目目录]/05_场景设定/[场景名]_设定卡.md`

### Step 4: 生成图片提示词

调用 `/drama-image` 技能生成：
- 全景空镜提示词
- 中景视角提示词
- 细节特写提示词
- 时段变体提示词

**输出路径**：`[项目目录]/06_场景图片/[场景名]_提示词.md`

### Step 5: 更新项目配置

将场景信息添加到 `00_项目配置.md` 的场景清单中。

## 示例

### 示例1：创建单个场景

```
输入：
/drama-scene 城市天台 --type outdoor --time dusk

输出：

📁 输出文件：
- 05_场景设定/城市天台_设定卡.md

📋 场景基本信息：
- 类型：室外场景
- 时段：黄昏
- 功能：主角独处、情绪释放

💡 下一步：
1. 查看并完善场景设定卡
```
