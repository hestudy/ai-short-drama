# 补充提示词

扫描所有分镜文件，检查并补充缺失的视频提示词。

## 用法

```
/drama-fill-prompts
```

## 前置条件

- 已运行 `/drama-storyboard` 生成分镜脚本
- 存在分镜文件待检查

## 执行流程

> ⚠️ **并发控制**：采用批量处理，每批 5 个镜头，批次间隔 1000ms

1. **扫描分镜文件**：读取所有 `03_分镜脚本/*.md` 文件
2. **识别缺失项**：
   - 提示词完全为空
   - 提示词过短（少于30字）
   - 提示词格式异常
3. **生成缺失报告**：列出需要补充的镜头清单
4. **批量补充**：每批 5 个镜头一起生成，减少 API 调用
5. **更新文件**：将补充的提示词写入分镜文件

## 检查规则

| 状态 | 判断条件 |
|------|----------|
| 缺失 | 提示词区域为空 |
| 异常 | 提示词少于30字 |
| 异常 | 提示词缺少运镜描述 |
| 异常 | 提示词缺少场景信息 |

## 输出示例

```markdown
# 提示词补充报告

## 扫描结果
- 扫描文件数：3 集
- 总镜头数：45 个
- 缺失提示词：5 个

## 需补充清单
| 集数 | 镜头 | 原因 |
|------|------|------|
| 第1集 | S01_C05 | 提示词为空 |
| 第2集 | S02_C03 | 缺少运镜描述 |
| 第2集 | S02_C08 | 提示词为空 |
| 第3集 | S03_C01 | 提示词过短 |
| 第3集 | S03_C11 | 提示词为空 |

## 补充完成
✅ 已为 5 个镜头补充提示词
```

## 使用场景

- 分镜生成后发现部分镜头缺少提示词
- 导出前进行完整性检查
- 审核前确保所有提示词完整

## 示例

```
/drama-fill-prompts
```
