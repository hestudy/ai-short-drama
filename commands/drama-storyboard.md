# 分镜脚本

将每集拆解为具体镜头，生成即梦AI视频提示词。

## 用法

```
/drama-storyboard           # 交互式选择集数
/drama-storyboard --all     # 生成所有集的分镜
/drama-storyboard 3         # 生成第3集分镜
/drama-storyboard 1-5       # 生成第1-5集分镜
```

## 前置条件

- 已运行 `/drama-character` 完成角色设计
- 存在角色外观提示词供参考

## 执行流程

> ⚠️ **并发控制**：采用批量处理，每批 5 个镜头，批次间隔 1000ms

1. 确认要处理的集数
2. 读取对应的故事大纲
3. 拆分镜头：
   - 场景划分
   - 镜头设计
   - 动作描述
   - 情绪标注
4. 批量为镜头生成视频提示词（每批 5 个）：
   ```
   [场景描述] + [主体特征与动作] + [视觉风格] + [情绪或氛围] + [镜头语言]
   ```
5. 写入分镜文件，每批完成后等待 1000ms

## 输出文件

```
03_分镜脚本/
└── 第X集_分镜.md

04_视频提示词/
└── 第X集_提示词.md
```

## 分镜格式

```markdown
## S01_C01 场景1-街道-日-外

### 镜头信息
- 场景：城市街道，黄昏
- 人物：林晓月
- 动作：疲惫地走在路上
- 情绪：孤独、疲惫
- 时长：3秒

### 画面描述
夕阳将天空染成橙红色，林晓月穿着职业套装，提着公文包，缓步走在人行道上。

### 即梦AI视频提示词
黄昏时分的城市街道，夕阳将天空染成橙红色。一位25岁的职场女性，穿着职业套装，疲惫地走在人行道上，偶尔抬头看看天，眼神中透露着孤独。温暖的橙色调，电影感构图，镜头从侧面中景跟随她缓慢移动。
```

## 示例

```
/drama-storyboard           # 交互式
/drama-storyboard --all     # 全部集数
/drama-storyboard 1         # 只做第1集
/drama-storyboard 3-5       # 第3到5集
```
