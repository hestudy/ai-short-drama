# 分镜脚本

将每集拆解为具体镜头，生成分镜表格。

> 💡 **v2.0 更新**：分镜脚本只生成分镜表格，视频提示词由 `/drama-prompt --batch` 独立生成

## 用法

```bash
/drama-storyboard           # 交互式选择集数
/drama-storyboard --all     # 生成所有集的分镜
/drama-storyboard 3         # 生成第3集分镜
/drama-storyboard 1-5       # 生成第1-5集分镜
```

## 前置条件

- 已运行 `/drama-plan` 完成故事大纲
- 已运行 `/drama-character` 完成角色设定
- 已运行 `/drama-scene` 完成场景设定
- 已生成角色参考图（`03_角色图片/`）
- 已生成场景参考图（`05_场景图片/`）

## 执行流程

### Step 1: 确认项目状态

1. 检查 `00_项目配置.md`
2. 读取角色清单和场景清单
3. 确认参考图已生成

### Step 2: 读取输入

1. 读取 `01_故事大纲.md`
2. 读取 `02_角色设定/` 目录
3. 读取 `04_场景设定/` 目录

### Step 3: 拆分镜头

为每集拆分镜头，包含：
- 场景编号和名称
- 镜头编号
- 景别
- 运镜
- 画面描述
- 角色
- 动作
- 情绪
- 台词（如有）
- 时长

### Step 4: 生成分镜表格

输出标准分镜表格到 `06_分镜脚本/`

## 输出文件

```
06_分镜脚本/
└── 第X集_分镜.md
```

## 分镜格式

```markdown
# 第1集 分镜脚本

## 集数信息
- 集数：第1集
- 标题：相遇
- 核心情节：[描述]
- 总时长：约60秒
- 镜头数：12个

## 角色索引
| 角色 | 参考图 |
|------|--------|
| 林晓月 | @林晓月_半身.jpg |
| 陈默 | @陈默_半身.jpg |

## 场景索引
| 场景 | 参考图 |
|------|--------|
| 城市天台 | @天台_全景_黄昏.jpg |
| 咖啡馆 | @咖啡馆_中景.jpg |

## 分镜表格

| 序号 | 场景 | 景别 | 运镜 | 画面描述 | 角色 | 动作 | 情绪 | 台词 | 时长 |
|------|------|------|------|----------|------|------|------|------|------|
| 01 | 天台黄昏 | 全景→中景 | 缓慢推近 | 夕阳下的城市天台，林晓月独自站在护栏边 | 林晓月 | 站立望向远方 | 孤独 | - | 3s |
| 02 | 天台黄昏 | 中景 | 固定 | 林晓月缓缓转身，眼眶含泪 | 林晓月 | 转身 | 悲伤 | 「我等你很久了。」 | 4s |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |

## 节奏分析
- 开场钩子：第01镜（0-3s）
- 情绪递进：第02-05镜
- 高潮点：第08镜
- 情感落点：第12镜
```

## 示例

```bash
/drama-storyboard           # 交互式选择
/drama-storyboard --all     # 生成所有集
/drama-storyboard 1         # 只做第1集
/drama-storyboard 3-5       # 第3到5集
```
