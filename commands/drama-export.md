# 导出提示词

汇总所有视频提示词、角色参考图、场景参考图，生成可直接使用的制作清单。

## 用法

```bash
/drama-export                      # 导出全部内容
/drama-export --format fusion      # 融合模式（默认）
/drama-export --format traditional # 传统模式
/drama-export --episode 3          # 只导出指定集数
```

## 前置条件

- 已完成至少一集的分镜和视频提示词
- 已生成角色参考图（`03_角色图片/`）
- 已生成场景参考图（`05_场景图片/`）

## 执行流程

### Step 1: 扫描项目

1. 读取 `00_项目配置.md`
2. 扫描 `03_角色图片/` 获取角色参考图列表
3. 扫描 `05_场景图片/` 获取场景参考图列表
4. 扫描 `07_视频提示词/` 获取所有视频提示词

### Step 2: 生成融合提示词包

为每个镜头生成完整的使用说明：

```
@角色参考图 @场景参考图

[视频提示词]

配音：
- 角色：xxx
- 台词：「xxx」
- 情绪：xxx

音频：
- 环境音：xxx
- 配乐：xxx
```

### Step 3: 生成参考图索引

```markdown
## 角色参考图索引
| 角色 | 文件路径 | 使用场景 |
|------|----------|----------|
| 林晓月 | 03_角色图片/林晓月/林晓月_半身.jpg | 第1-5集 |
| 陈默 | 03_角色图片/陈默/陈默_半身.jpg | 第2-5集 |

## 场景参考图索引
| 场景 | 文件路径 | 使用场景 |
|------|----------|----------|
| 城市天台 | 05_场景图片/天台/天台_全景_黄昏.jpg | 第1集 |
| 咖啡馆 | 05_场景图片/咖啡馆/咖啡馆_中景.jpg | 第2-3集 |
```

### Step 4: 生成制作清单

## 输出文件

```
99_导出包/
├── 融合提示词.md           # Seedance 2.0 专用，直接复制到即梦AI
├── 角色参考图索引.md       # 角色参考图对照表
├── 场景参考图索引.md       # 场景参考图对照表
└── 制作清单.md             # 素材制作进度追踪表
```

## 与即梦AI配合

1. **准备参考图**
   - 打开 `99_导出包/` 目录
   - 将 `03_角色图片/` 和 `05_场景图片/` 中的图片上传到即梦AI

2. **生成视频**
   - 打开 `融合提示词.md`
   - 复制镜头提示词（包含 `@参考图` 语法）
   - 粘贴到即梦AI
   - 一键生成带配音视频

3. **后期剪辑**
   - 下载所有视频片段
   - 使用剪辑软件整合

## 示例

```bash
/drama-export                    # 导出全部内容（融合模式）
/drama-export --episode 1-3      # 只导出前3集
```
