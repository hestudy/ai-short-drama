# 导出提示词

汇总所有视频提示词和配音文本，生成可直接使用的制作清单。

## 用法

```
/drama-export
/drama-export --format [格式]   # 选择导出格式
/drama-export --episode 3       # 只导出指定集数
```

## 前置条件

- 已完成至少一集的分镜和提示词
- 建议完成配音台本

## 执行流程

1. 扫描项目目录，收集所有内容
2. 汇总视频提示词：
   - 按集数和镜头排序
   - 添加镜头编号
   - 标注时长
3. 汇总角色提示词：
   - 角色外观提示词
   - 参考图说明
4. 汇总配音文本：
   - 按集数整理
   - 标注角色和情绪
5. 生成制作清单

## 输出文件

```
99_导出包/
├── 全部提示词.md           # 视频提示词汇总，直接复制到即梦AI
├── 角色提示词.md           # 角色外观提示词汇总
├── 配音文本.md             # 配音台本汇总
└── 制作清单.md             # 素材制作进度追踪表
```

## 导出格式示例

### 全部提示词.md


```

---

## 第2集
...
```

## 与即梦AI配合

1. **生成角色参考图**
   - 打开 `角色提示词.md`
   - 复制角色提示词到即梦AI
   - 生成满意的角色图

2. **生成视频片段**
   - 打开 `全部提示词.md`
   - 逐个复制镜头提示词
   - 上传对应角色参考图
   - 生成视频

3. **配音**
   - 打开 `配音文本.md`
   - 复制到即梦AI配音
   - 下载音频文件

## 示例

```
/drama-export                    # 导出全部内容
/drama-export --episode 1-3      # 只导出前3集
```

## 工作流完成

导出完成后，整个AI短剧文字工作流结束。可以开始视频制作阶段！
