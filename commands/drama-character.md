# 角色设计

创建角色设定卡，生成即梦AI图片提示词。

## 用法

```
/drama-character
/drama-character --all        # 一次性生成所有角色
/drama-character [角色名]     # 生成/更新指定角色
```

## 前置条件

- 已运行 `/drama-plan` 生成故事大纲
- 故事大纲中包含角色信息

## 执行流程

> ⚠️ **并发控制**：角色间需保持风格一致，采用顺序处理，每个角色完成后等待 1000ms

1. 从故事大纲提取角色清单
2. 逐个角色进行设计：
   - 基本信息（姓名、年龄、身份）
   - 性格特点
   - 外貌特征
   - 服装造型
   - 人物关系
3. 批量生成即梦AI图片提示词（一次生成所有表情）：
   ```
   [人物外貌] + [服装描述] + [表情姿态] + [风格] + [光影]
   ```
4. 写入角色文件，等待 1000ms 后处理下一角色

## 输出文件

```
02_角色设定/
├── [角色名]_设定卡.md      # 角色详细信息
└── [角色名]_外观提示词.md  # 即梦AI图片提示词
```

- 标志性特征：

- 特殊装扮：

[可直接复制使用的提示词]

## 示例

```
/drama-character           # 交互式创建角色
/drama-character --all     # 一次性生成所有角色
/drama-character 林晓月    # 只生成林晓月角色
```
