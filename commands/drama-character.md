# 角色设计

创建角色设定卡，定义一致性控制词。

> 💡 **v2.0 更新**：角色设定卡只定义角色信息和一致性控制词，图片提示词由 `/drama-image --character` 独立生成

## 用法

```bash
/drama-character              # 交互式创建角色
/drama-character --all        # 一次性生成所有角色
/drama-character [角色名]     # 生成/更新指定角色
```

## 前置条件

- 已运行 `/drama-plan` 生成故事大纲
- 故事大纲中包含角色信息

## 执行流程

### Step 1: 提取角色清单

从 `01_故事大纲.md` 中提取所有角色信息

### Step 2: 创建角色设定卡

使用 `references/character-card.md` 模板，为每个角色创建设定卡：

1. **基本信息**
   - 姓名、年龄、身份
   - 在故事中的功能

2. **外貌特征**
   - 脸型、发型、体型
   - 标志性特征（重要！用于一致性控制）

3. **服装造型**
   - 常规装扮
   - 特殊装扮

4. **性格与情绪**
   - 性格关键词
   - 常见表情
   - 习惯动作

5. **一致性控制词**（核心！）
   - 固定特征词：每次生成都必须包含
   - 禁止变化项：不允许改变的特征

### Step 3: 更新项目配置

将角色信息添加到 `00_项目配置.md` 的角色清单中

```markdown
## 一致性控制词

### 固定特征词（每次必含）
```
[年龄]的[性别]，[发型发色]，[面部特征]，
[标志性特征]，[体型描述]，...
```

### 禁止变化项
- [ ] 发型
- [ ] 标志性特征
- [ ] 体型

## 出现镜头记录
| 集数 | 镜头 | 情绪 | 备注 |
|------|------|------|------|
```

## 示例

```bash
/drama-character              # 交互式创建角色
/drama-character --all        # 一次性生成所有角色
/drama-character 林晓月       # 只生成林晓月角色
```
