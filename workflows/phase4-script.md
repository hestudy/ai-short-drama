# 阶段四：配音台本工作流

## 目标

生成完整的配音台本，可直接用于即梦AI的语音合成功能。

## 输入

- `03_分镜脚本/`

## 输出

- `05_配音台本/第X集.md` × N
- `99_导出包/配音文本.md`

---

## 执行步骤

### Step 1: 确认配音风格

与用户确认：

```
配音风格选择：

1. 【配音类型】
   - 全配音：所有角色都有台词配音
   - 旁白为主：主要是旁白叙述，少量角色台词
   - 混合型：旁白+角色对话

2. 【旁白风格】（如需要）
   - 第一人称：以主角视角叙述
   - 第三人称：客观叙述
   - 无旁白：纯对话推进

3. 【角色声音特点】
   - 请描述各角色的声音特点偏好

4. 【语言风格】
   - 正式/口语化
   - 现代感/古风
   - 特殊要求
```

### Step 2: 生成配音台本

对每一集：

```markdown

| 角色A | 温柔女声，中速 | [建议选项] |
| 角色B | 沉稳男声，偏慢 | [建议选项] |
```

### Step 3: 生成汇总导出文件

创建 `99_导出包/配音文本.md`：

```markdown

---

## 第1集

### 角色声音配置
- 角色A：[即梦AI声音选项]
- 角色B：[即梦AI声音选项]
- 旁白：[即梦AI声音选项]

### 台词文本
```
[00:00] [角色A] 台词内容
[00:05] [旁白] 台词内容
[00:12] [角色B] 台词内容
...
```

---

## 第2集

（同上格式）

## 使用说明

1. 在即梦AI中选择对应角色声音
2. 复制台词文本进行配音
3. 根据情绪标注调整配音参数
4. 生成后检查与画面的同步
```

---

## 台词写作原则

```
✅ 短小精悍：单句不超过20字
✅ 口语化：符合日常说话习惯
✅ 情绪明确：台词本身带有情绪色彩
✅ 信息高效：每句话都有叙事功能
✅ 节奏感：长短句交替，有呼吸感

❌ 避免书面化表达
❌ 避免过长的独白
❌ 避免信息重复
❌ 避免与画面信息完全重叠
```
